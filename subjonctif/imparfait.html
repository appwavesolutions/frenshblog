Qu'est-ce que l'imparfait ?

L'imparfait est un temps du passé qui exprime :

  • Une action en cours dans le passé : "Je lisais un livre"
  • Une habitude passée : "Tous les dimanches, nous allions au parc"
  • Une description dans le passé : "Il faisait beau, le soleil brillait"
  • Un état dans le passé : "J'étais fatigué"

Exemples d'usage

Exemples courants :

  • Action en cours : "Hier, je regardais la télévision"
  • Habitude : "Quand j'étais petit, je jouais au football"
  • Description : "Le ciel était bleu, les oiseaux chantaient"
  • État : "Elle était très contente de te voir"

Terminaisons de l'imparfait

L'imparfait se forme avec le radical du verbe + les terminaisons spécifiques.

Pronom Terminaison Exemple
Je -ais je parlais
Tu -ais tu parlais
Il/Elle/On -ait il parlait
Nous -ions nous parlions
Vous -iez vous parliez
Ils/Elles -aient ils parlaient

Construction par groupe de verbes

1er groupe (verbes en -er)

Radical : infinitif sans -er + terminaisons

Exemple : parler

  • Je parlais
  • Tu parlais
  • Il parlait
  • Nous parlions
  • Vous parliez
  • Ils parlaient

2ème groupe (verbes en -ir)

Radical : infinitif sans -ir + terminaisons

Exemple : finir

  • Je finissais
  • Tu finissais
  • Il finissait
  • Nous finissions
  • Vous finissiez
  • Ils finissaient

3ème groupe (verbes irréguliers)

Radical : forme spécifique + terminaisons

Exemples :

  • Être : j'étais, tu étais, il était, nous étions, vous étiez, ils étaient
  • Avoir : j'avais, tu avais, il avait, nous avions, vous aviez, ils avaient
  • Aller : j'allais, tu allais, il allait, nous allions, vous alliez, ils allaient
  • Faire : je faisais, tu faisais, il faisait, nous faisions, vous faisiez, ils faisaient
  • Venir : je venais, tu venais, il venait, nous venions, vous veniez, ils venaient

Emplois de l'imparfait

1. Action en cours dans le passé

Quand une action était en train de se dérouler

  • "Hier à 15h, je regardais la télévision"
  • "Pendant que tu dormais, je travaillais"
  • "Il pleuvait quand nous sommes sortis"

2. Habitudes passées

  • Fréquence : "Tous les dimanches, nous allions au parc"
  • Répétition : "Quand j'étais petit, je jouais au football"
  • Routine : "Chaque matin, je prenais le bus"

3. Description dans le passé

  • Lieu : "La maison était grande et blanche"
  • Personne : "Marie était très belle"
  • Atmosphère : "Il faisait beau, le soleil brillait"

4. État dans le passé

  • Sentiments : "J'étais content de te voir"
  • Conditions : "Il était malade hier"
  • Possession : "Nous avions une voiture rouge"

Imparfait vs Passé composé

Imparfait Passé composé
Action en cours Action terminée
Habitudes Événements précis
Description Résultat
"Je lisais" "J'ai lu"

Exercices pratiques

Exercice 1 : Conjuguez à l'imparfait

  1. Je (manger) ___________ une pomme
  2. Tu (finir) ___________ tes devoirs
  3. Il (être) ___________ fatigué
  4. Nous (avoir) ___________ faim
  5. Vous (aller) ___________ au cinéma
  6. Ils (faire) ___________ leurs courses

Exercice 2 : Choisissez entre imparfait et passé composé

  1. Hier, je (regarder) ___________ la télé quand tu (arriver) ___________
  2. Quand j'étais petit, je (jouer) ___________ au football
  3. Il (pleuvoir) ___________ quand nous (sortir) ___________
  4. Marie (être) ___________ très contente de te voir

Exercice 3 : Transformez en phrases à l'imparfait

  1. Je mange une pomme → ___________
  2. Il fait beau → ___________
  3. Nous allons au parc → ___________
  4. Tu es content → ___________

Conseils pour bien utiliser l'imparfait

Points clés à retenir :

  • L'imparfait décrit des actions en cours, des habitudes ou des états dans le passé
  • Les terminaisons sont toujours les mêmes : -ais, -ais, -ait, -ions, -iez, -aient
  • Utilisez l'imparfait pour la description et le passé composé pour les actions terminées
  • Les verbes être et avoir sont irréguliers : j'étais, j'avais
  • L'imparfait exprime souvent la durée, le passé composé exprime le résultat
