Qu'est-ce que le passé composé ?

Le passé composé est un temps du passé qui exprime :

  • Une action terminée dans le passé : "J'ai mangé une pomme"
  • Un événement précis : "Hier, il a plu"
  • Un résultat : "J'ai fini mes devoirs"
  • Une action ponctuelle : "Il est arrivé à 8h"

Exemples d'usage

Exemples courants :

  • Action terminée : "J'ai lu ce livre hier"
  • Événement précis : "Marie est née en 1990"
  • Résultat : "Nous avons gagné le match"
  • Action ponctuelle : "Il a téléphoné à 10h"

Construction du passé composé

Le passé composé se forme avec : Auxiliaire (avoir/être) + Participe passé

Auxiliaires avoir et être

Pronom AVOIR ÊTRE
Je j'ai je suis
Tu tu as tu es
Il/Elle/On il a il est
Nous nous avons nous sommes
Vous vous avez vous êtes
Ils/Elles ils ont ils sont

Participes passés

Le participe passé se forme différemment selon le groupe du verbe :

1er groupe (verbes en -er)

Règle : infinitif + é

  • parler → parlé
  • manger → mangé
  • regarder → regardé

2ème groupe (verbes en -ir)

Règle : infinitif + i

  • finir → fini
  • choisir → choisi
  • réussir → réussi

3ème groupe (verbes irréguliers)

Exemples courants :

  • être → été
  • avoir → eu
  • faire → fait
  • prendre → pris
  • venir → venu
  • aller → allé

Verbes avec AVOIR

La plupart des verbes utilisent l'auxiliaire AVOIR au passé composé.

Exemples avec AVOIR :

  • J'ai mangé une pomme
  • Tu as regardé la télévision
  • Il a fini ses devoirs
  • Nous avons joué au football
  • Vous avez travaillé dur
  • Ils ont voyagé en France

Verbes avec ÊTRE

Certains verbes utilisent l'auxiliaire ÊTRE. Ce sont principalement :

Les verbes de mouvement

  • aller → Je suis allé(e)
  • venir → Tu es venu(e)
  • arriver → Il est arrivé
  • partir → Nous sommes partis
  • sortir → Vous êtes sorti(e)s
  • entrer → Ils sont entrés

Les verbes d'état

  • naître → Je suis né(e)
  • mourir → Il est mort
  • devenir → Elle est devenue
  • rester → Nous sommes restés

Mémo technique : DR & MRS VANDERTRAMP

Devenir, Rester, Monter, Retourner, Sortir

Venir, Aller, Naître, Descendre, Entrer, Rentrer, Tomber, Repartir, Arriver, Mourir, Partir

Accords du participe passé

Avec l'auxiliaire AVOIR

Le participe passé ne s'accorde PAS avec le sujet, mais avec le COD si celui-ci est placé AVANT le verbe.

Exemples :

  • Pas d'accord : "J'ai mangé une pomme" (COD après)
  • Avec accord : "La pomme que j'ai mangée" (COD avant)
  • Pas d'accord : "Nous avons vu un film" (COD après)
  • Avec accord : "Le film que nous avons vu" (COD avant)

Avec l'auxiliaire ÊTRE

Le participe passé s'accorde TOUJOURS avec le sujet.

Exemples :

  • Je suis allé(e) au cinéma
  • Tu es venu(e) hier
  • Il est arrivé à l'heure
  • Elle est partie tôt
  • Nous sommes restés à la maison
  • Vous êtes sorti(e)s hier soir
  • Ils sont entrés dans la salle
  • Elles sont venues nous voir

Emplois du passé composé

1. Action terminée dans le passé

  • Événement précis : "Hier, j'ai acheté un livre"
  • Action ponctuelle : "Il a téléphoné à 10h"
  • Résultat : "J'ai fini mes devoirs"

2. Série d'actions dans le passé

  • "Je me suis levé, j'ai pris ma douche, j'ai déjeuné"
  • "Il est arrivé, il a salué tout le monde, il s'est assis"

3. Action passée avec conséquence dans le présent

  • "J'ai perdu mes clés" (et je ne les ai toujours pas)
  • "Il a cassé son téléphone" (et il n'en a plus)

Imparfait vs Passé composé

Imparfait Passé composé
Action en cours Action terminée
Habitudes Événements précis
Description Résultat
"Je lisais" "J'ai lu"
"Il pleuvait" "Il a plu"

Exercices pratiques

Exercice 1 : Conjuguez au passé composé

  1. Je (manger) ___________ une pomme
  2. Tu (finir) ___________ tes devoirs
  3. Il (être) ___________ content
  4. Nous (avoir) ___________ faim
  5. Vous (aller) ___________ au cinéma
  6. Ils (faire) ___________ leurs courses

Exercice 2 : Choisissez l'auxiliaire correct

  1. Je (avoir/être) ___________ mangé une pomme
  2. Tu (avoir/être) ___________ allé au cinéma
  3. Il (avoir/être) ___________ né en 1990
  4. Nous (avoir/être) ___________ regardé un film
  5. Vous (avoir/être) ___________ sorti hier soir
  6. Ils (avoir/être) ___________ fait leurs devoirs

Exercice 3 : Accordez le participe passé

  1. La pomme que j'ai mangé(e) → ___________
  2. Marie est allé(e) au parc → ___________
  3. Les livres qu'il a lu(s) → ___________
  4. Elles sont venu(e)s nous voir → ___________

Exercice 4 : Choisissez entre imparfait et passé composé

  1. Hier, je (regarder) ___________ la télé quand tu (arriver) ___________
  2. Quand j'étais petit, je (jouer) ___________ au football
  3. Il (pleuvoir) ___________ quand nous (sortir) ___________
  4. Marie (être) ___________ très contente de te voir

Conseils pour bien utiliser le passé composé

Points clés à retenir :

  • Le passé composé = Auxiliaire (avoir/être) + Participe passé
  • La plupart des verbes utilisent AVOIR
  • Les verbes de mouvement et d'état utilisent ÊTRE
  • Avec AVOIR : accord avec le COD si placé avant
  • Avec ÊTRE : accord toujours avec le sujet
  • Utilisez le passé composé pour des actions terminées
  • Mémoire : DR & MRS VANDERTRAMP pour les verbes avec ÊTRE
