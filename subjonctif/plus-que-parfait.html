Qu'est-ce que le plus-que-parfait ?

Le plus-que-parfait exprime :

  • Une action antérieure à une autre action passée : "J'avais mangé quand tu es arrivé"
  • Un état antérieur dans le passé : "Il était déjà parti"
  • Une action accomplie avant un moment du passé : "J'avais fini mes devoirs"
  • Un regret dans le passé : "J'aurais dû partir plus tôt"

Exemples d'usage

Exemples courants :

  • Antériorité : "Quand je suis arrivé, il avait déjà mangé"
  • État antérieur : "Elle était déjà partie quand j'ai téléphoné"
  • Action accomplie : "J'avais fini mes devoirs avant de sortir"
  • Regret : "J'aurais dû l'écouter"

Construction du plus-que-parfait

Le plus-que-parfait se forme avec : Auxiliaire à l'imparfait + Participe passé

Formation générale

Structure :

  • AVOIR à l'imparfait + participe passé
  • ÊTRE à l'imparfait + participe passé

Exemple : J'avais mangé / Je étais allé

Auxiliaires à l'imparfait

Pronom AVOIR ÊTRE
Je j'avais j'étais
Tu tu avais tu étais
Il/Elle/On il avait il était
Nous nous avions nous étions
Vous vous aviez vous étiez
Ils/Elles ils avaient ils étaient

Verbes avec AVOIR

La plupart des verbes utilisent l'auxiliaire AVOIR au plus-que-parfait.

Exemples avec AVOIR :

  • J'avais mangé une pomme
  • Tu avais regardé la télévision
  • Il avait fini ses devoirs
  • Nous avions joué au football
  • Vous aviez travaillé dur
  • Ils avaient voyagé en France

Verbes avec ÊTRE

Les mêmes verbes qu'au passé composé utilisent ÊTRE au plus-que-parfait.

Les verbes de mouvement

  • aller → J'étais allé(e)
  • venir → Tu étais venu(e)
  • arriver → Il était arrivé
  • partir → Nous étions partis
  • sortir → Vous étiez sorti(e)s
  • entrer → Ils étaient entrés

Les verbes d'état

  • naître → J'étais né(e)
  • mourir → Il était mort
  • devenir → Elle était devenue
  • rester → Nous étions restés

Accords du participe passé

Avec l'auxiliaire AVOIR

Le participe passé ne s'accorde PAS avec le sujet, mais avec le COD si celui-ci est placé AVANT le verbe.

Exemples :

  • Pas d'accord : "J'avais mangé une pomme" (COD après)
  • Avec accord : "La pomme que j'avais mangée" (COD avant)
  • Pas d'accord : "Nous avions vu un film" (COD après)
  • Avec accord : "Le film que nous avions vu" (COD avant)

Avec l'auxiliaire ÊTRE

Le participe passé s'accorde TOUJOURS avec le sujet.

Exemples :

  • J'étais allé(e) au cinéma
  • Tu étais venu(e) hier
  • Il était arrivé à l'heure
  • Elle était partie tôt
  • Nous étions restés à la maison
  • Vous étiez sorti(e)s hier soir
  • Ils étaient entrés dans la salle
  • Elles étaient venues nous voir

Emplois du plus-que-parfait

1. Antériorité dans le passé

Action antérieure à une autre action passée

  • "Quand je suis arrivé, il avait déjà mangé"
  • "J'avais fini mes devoirs avant qu'il ne vienne"
  • "Elle était déjà partie quand j'ai téléphoné"

2. État antérieur

  • État accompli : "Il était déjà marié"
  • Action terminée : "J'avais déjà vu ce film"
  • Expérience passée : "Nous avions déjà visité Paris"

3. Cause dans le passé

  • Explication : "Il était fatigué parce qu'il avait mal dormi"
  • Conséquence : "J'étais en retard car j'avais raté le bus"

4. Regret ou reproche

  • Regret : "J'aurais dû partir plus tôt"
  • Reproche : "Tu aurais pu me prévenir"
  • Reproche : "Il aurait fallu réserver"

Chronologie des temps

Temps Usage Exemple
Plus-que-parfait Action la plus ancienne J'avais mangé
Imparfait Action en cours Je regardais
Passé composé Action la plus récente Tu es arrivé

Exercices pratiques

Exercice 1 : Conjuguez au plus-que-parfait

  1. Je (manger) ___________ une pomme
  2. Tu (finir) ___________ tes devoirs
  3. Il (être) ___________ content
  4. Nous (avoir) ___________ faim
  5. Vous (aller) ___________ au cinéma
  6. Ils (faire) ___________ leurs courses

Exercice 2 : Choisissez l'auxiliaire correct

  1. Je (avoir/être) ___________ mangé une pomme
  2. Tu (avoir/être) ___________ allé au cinéma
  3. Il (avoir/être) ___________ né en 1990
  4. Nous (avoir/être) ___________ regardé un film
  5. Vous (avoir/être) ___________ sorti hier soir
  6. Ils (avoir/être) ___________ fait leurs devoirs

Exercice 3 : Accordez le participe passé

  1. La pomme que j'avais mangé(e) → ___________
  2. Marie était allé(e) au parc → ___________
  3. Les livres qu'il avait lu(s) → ___________
  4. Elles étaient venu(e)s nous voir → ___________

Exercice 4 : Complétez avec le bon temps

  1. Quand je suis arrivé, il (manger) ___________ déjà
  2. J'(finir) ___________ mes devoirs avant qu'il ne vienne
  3. Elle (partir) ___________ quand j'ai téléphoné
  4. Nous (voir) ___________ ce film plusieurs fois

Conseils pour bien utiliser le plus-que-parfait

Points clés à retenir :

  • Le plus-que-parfait = Auxiliaire à l'imparfait + Participe passé
  • Il exprime l'antériorité dans le passé
  • La plupart des verbes utilisent AVOIR
  • Les verbes de mouvement et d'état utilisent ÊTRE
  • Avec AVOIR : accord avec le COD si placé avant
  • Avec ÊTRE : accord toujours avec le sujet
  • Utilisez-le pour exprimer une action antérieure à une autre action passée
