Qu'est-ce que le conditionnel ?

Le conditionnel est un mode qui exprime :

  • L'hypothèse : "Si j'avais de l'argent, je voyagerais"
  • La politesse : "Je voudrais un café, s'il vous plaît"
  • Le futur dans le passé : "Il a dit qu'il viendrait"
  • Le regret : "J'aurais dû partir plus tôt"

Exemples d'usage

Exemples courants :

  • Hypothèse : "Si j'étais riche, j'achèterais une maison"
  • Politesse : "Pourriez-vous m'aider ?"
  • Futur dans le passé : "Marie pensait qu'il pleuvrait"
  • Conseil : "Tu devrais étudier plus"

Conditionnel présent

Le conditionnel présent se forme avec le radical du futur + les terminaisons de l'imparfait.

Terminaisons du conditionnel présent

Pronom Terminaison Exemple
Je -ais je parlerais
Tu -ais tu parlerais
Il/Elle/On -ait il parlerait
Nous -ions nous parlerions
Vous -iez vous parleriez
Ils/Elles -aient ils parleraient

Construction par groupe de verbes

1er groupe (verbes en -er)

Radical : infinitif sans -er + terminaisons

Exemple : parler

  • Je parlerais
  • Tu parlerais
  • Il parlerait
  • Nous parlerions
  • Vous parleriez
  • Ils parleraient

2ème groupe (verbes en -ir)

Radical : infinitif sans -ir + terminaisons

Exemple : finir

  • Je finirais
  • Tu finirais
  • Il finirait
  • Nous finirions
  • Vous finiriez
  • Ils finiraient

3ème groupe (verbes irréguliers)

Radical : futur simple + terminaisons

Exemples :

  • Être : je serais, tu serais, il serait...
  • Avoir : j'aurais, tu aurais, il aurait...
  • Aller : j'irais, tu irais, il irait...
  • Faire : je ferais, tu ferais, il ferait...

Conditionnel passé

Le conditionnel passé se forme avec l'auxiliaire (avoir ou être) au conditionnel présent + participe passé.

Construction du conditionnel passé

Auxiliaire Conditionnel présent + Participe passé
Avoir j'aurais + parlé
Être je serais + allé(e)

Exemples par groupe

Avec l'auxiliaire AVOIR

Exemple : parler

  • J'aurais parlé
  • Tu aurais parlé
  • Il aurait parlé
  • Nous aurions parlé
  • Vous auriez parlé
  • Ils auraient parlé

Avec l'auxiliaire ÊTRE

Exemple : aller

  • Je serais allé(e)
  • Tu serais allé(e)
  • Il serait allé
  • Nous serions allé(e)s
  • Vous seriez allé(e)s
  • Ils seraient allés

Note : Accord avec le sujet pour les verbes avec être

Les trois types de propositions conditionnelles

Les propositions conditionnelles expriment une condition et sa conséquence. En français, il existe trois types principaux :

Type 1 : Condition probable (Réalité possible)

Structure : Si + Présent + Futur/Présent

Usage : Exprime une condition réelle ou probable dans le futur

Exemples :

  • "Si je gagne à la loterie, je partirai en voyage"
  • "Si tu étudies bien, tu réussiras ton examen"
  • "Si il pleut demain, nous resterons à la maison"
  • "Si vous venez, nous serons heureux"

Astuce : La condition est réalisable, probable ou sur le point de se réaliser.

Type 2 : Condition irréelle ou peu probable (Hypothèse)

Structure : Si + Imparfait + Conditionnel Présent

Usage : Exprime une condition irréelle, hypothétique ou peu probable dans le présent ou le futur

Exemples :

  • "Si j'étais riche, je voyagerais dans le monde entier"
  • "Si tu avais plus de temps, tu pourrais lire davantage"
  • "Si nous habitions en France, nous parlerions mieux français"
  • "Si vous vouliez, vous feriez des progrès"

Astuce : La condition est peu probable, hypothétique, ou exprime un souhait irréalisable dans le présent.

Type 3 : Condition impossible (Regret)

Structure : Si + Plus-que-parfait + Conditionnel Passé

Usage : Exprime une condition totalement irréalisable dans le passé et son regret

Exemples :

  • "Si j'avais su, je serais venu t'aider"
  • "Si tu avais étudié plus, tu aurais réussi ton examen"
  • "Si nous étions partis plus tôt, nous serions arrivés à l'heure"
  • "Si vous m'aviez appelé, je vous aurais rejoint"

Astuce : Exprime le regret d'une action passée qui n'a pas eu lieu. Impossible à changer maintenant.

Tableau récapitulatif

Type Si + Temps Conséquence Exemple
Type 1
Probable		 Présent Futur / Présent Si j'ai le temps, je viendrai
Type 2
Improbable		 Imparfait Conditionnel présent Si j'avais le temps, je viendrais
Type 3
Impossible		 Plus-que-parfait Conditionnel passé Si j'avais eu le temps, je serais venu

Erreurs fréquentes à éviter

  • Ne pas confondre : "Si j'aurais su" → "Si j'avais su"
  • Le type 2 et le type 3 ne peuvent PAS être mélangés
  • Respecter la concordance des temps : chaque type a sa structure fixe
  • Attention aux auxiliaires : avoir/être au plus-que-parfait dans le type 3

Emplois du conditionnel

1. L'hypothèse

Si + imparfait + conditionnel présent

  • "Si j'avais de l'argent, je voyagerais"
  • "Si tu étudiais plus, tu réussirais"
  • "Si nous partions tôt, nous arriverions à l'heure"

2. La politesse

  • Demandes : "Je voudrais un café"
  • Questions : "Pourriez-vous m'aider ?"
  • Suggestions : "Il faudrait peut-être partir"

3. Le futur dans le passé

  • "Il a dit qu'il viendrait demain"
  • "Je pensais qu'elle serait là"
  • "Nous savions qu'il pleuvrait"

4. Le regret

  • "J'aurais dû partir plus tôt"
  • "Tu aurais pu me prévenir"
  • "Il aurait fallu réserver"

Exercices pratiques

Exercice 1 : Conjuguez au conditionnel présent

  1. Je (parler) ___________ avec lui
  2. Tu (finir) ___________ tes devoirs
  3. Il (être) ___________ content
  4. Nous (avoir) ___________ le temps
  5. Vous (aller) ___________ au cinéma
  6. Ils (faire) ___________ leurs courses

Exercice 2 : Complétez avec le conditionnel passé

  1. Si j'avais su, je (venir) ___________
  2. Tu (devoir) ___________ m'appeler
  3. Elle (pouvoir) ___________ nous aider
  4. Nous (vouloir) ___________ partir plus tôt

Exercice 3 : Transformez en phrases polies

  1. Donne-moi un verre d'eau → ___________
  2. Peux-tu m'aider ? → ___________
  3. Je veux un café → ___________
  4. Il faut partir maintenant → ___________

Exercice 4 : Complétez avec le bon type de proposition conditionnelle

Identifiez le type et complétez :

  1. Si j'___________ (avoir) de l'argent, je ___________ (acheter) une voiture. (Type ?)
  2. Si tu ___________ (venir) demain, nous ___________ (être) contents. (Type ?)
  3. Si tu ___________ (étudier) plus, tu ___________ (réussir) ton examen. (Type ?)
  4. Si il ___________ (pleuvoir) hier, nous ___________ (rester) à la maison. (Type ?)
  5. Si je ___________ (pouvoir), je ___________ (t'aider). (Type ?)
  6. Si vous ___________ (m'appeler) hier, je ___________ (venir) vous voir. (Type ?)

Conseils pour bien utiliser le conditionnel

Points clés à retenir :

  • Formation : Le conditionnel présent = radical du futur + terminaisons de l'imparfait
  • Formation : Le conditionnel passé = auxiliaire au conditionnel + participe passé
  • Les 3 types de propositions :
    • Type 1 (probable) : Si + Présent → Futur
    • Type 2 (hypothétique) : Si + Imparfait → Conditionnel présent
    • Type 3 (impossible/passé) : Si + Plus-que-parfait → Conditionnel passé
  • Usage : Utilisez-le pour l'hypothèse, la politesse et le futur dans le passé
  • Accord : N'oubliez pas l'accord avec le sujet pour les verbes avec être
  • Erreur fréquente : Ne dites jamais "Si j'aurais su" → Utilisez "Si j'avais su"
