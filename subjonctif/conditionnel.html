Qu'est-ce que le conditionnel ?

Le conditionnel est un mode qui exprime :

  • L'hypothèse : "Si j'avais de l'argent, je voyagerais"
  • La politesse : "Je voudrais un café, s'il vous plaît"
  • Le futur dans le passé : "Il a dit qu'il viendrait"
  • Le regret : "J'aurais dû partir plus tôt"

Exemples d'usage

Exemples courants :

  • Hypothèse : "Si j'étais riche, j'achèterais une maison"
  • Politesse : "Pourriez-vous m'aider ?"
  • Futur dans le passé : "Marie pensait qu'il pleuvrait"
  • Conseil : "Tu devrais étudier plus"

Conditionnel présent

Le conditionnel présent se forme avec le radical du futur + les terminaisons de l'imparfait.

Terminaisons du conditionnel présent

Pronom Terminaison Exemple
Je -ais je parlerais
Tu -ais tu parlerais
Il/Elle/On -ait il parlerait
Nous -ions nous parlerions
Vous -iez vous parleriez
Ils/Elles -aient ils parleraient

Construction par groupe de verbes

1er groupe (verbes en -er)

Radical : infinitif sans -er + terminaisons

Exemple : parler

  • Je parlerais
  • Tu parlerais
  • Il parlerait
  • Nous parlerions
  • Vous parleriez
  • Ils parleraient

2ème groupe (verbes en -ir)

Radical : infinitif sans -ir + terminaisons

Exemple : finir

  • Je finirais
  • Tu finirais
  • Il finirait
  • Nous finirions
  • Vous finiriez
  • Ils finiraient

3ème groupe (verbes irréguliers)

Radical : futur simple + terminaisons

Exemples :

  • Être : je serais, tu serais, il serait...
  • Avoir : j'aurais, tu aurais, il aurait...
  • Aller : j'irais, tu irais, il irait...
  • Faire : je ferais, tu ferais, il ferait...

Conditionnel passé

Le conditionnel passé se forme avec l'auxiliaire (avoir ou être) au conditionnel présent + participe passé.

Construction du conditionnel passé

Auxiliaire Conditionnel présent + Participe passé
Avoir j'aurais + parlé
Être je serais + allé(e)

Exemples par groupe

Avec l'auxiliaire AVOIR

Exemple : parler

  • J'aurais parlé
  • Tu aurais parlé
  • Il aurait parlé
  • Nous aurions parlé
  • Vous auriez parlé
  • Ils auraient parlé

Avec l'auxiliaire ÊTRE

Exemple : aller

  • Je serais allé(e)
  • Tu serais allé(e)
  • Il serait allé
  • Nous serions allé(e)s
  • Vous seriez allé(e)s
  • Ils seraient allés

Note : Accord avec le sujet pour les verbes avec être

Emplois du conditionnel

1. L'hypothèse

Si + imparfait + conditionnel présent

  • "Si j'avais de l'argent, je voyagerais"
  • "Si tu étudiais plus, tu réussirais"
  • "Si nous partions tôt, nous arriverions à l'heure"

2. La politesse

  • Demandes : "Je voudrais un café"
  • Questions : "Pourriez-vous m'aider ?"
  • Suggestions : "Il faudrait peut-être partir"

3. Le futur dans le passé

  • "Il a dit qu'il viendrait demain"
  • "Je pensais qu'elle serait là"
  • "Nous savions qu'il pleuvrait"

4. Le regret

  • "J'aurais dû partir plus tôt"
  • "Tu aurais pu me prévenir"
  • "Il aurait fallu réserver"

Exercices pratiques

Exercice 1 : Conjuguez au conditionnel présent

  1. Je (parler) ___________ avec lui
  2. Tu (finir) ___________ tes devoirs
  3. Il (être) ___________ content
  4. Nous (avoir) ___________ le temps
  5. Vous (aller) ___________ au cinéma
  6. Ils (faire) ___________ leurs courses

Exercice 2 : Complétez avec le conditionnel passé

  1. Si j'avais su, je (venir) ___________
  2. Tu (devoir) ___________ m'appeler
  3. Elle (pouvoir) ___________ nous aider
  4. Nous (vouloir) ___________ partir plus tôt

Exercice 3 : Transformez en phrases polies

  1. Donne-moi un verre d'eau → ___________
  2. Peux-tu m'aider ? → ___________
  3. Je veux un café → ___________
  4. Il faut partir maintenant → ___________

Conseils pour bien utiliser le conditionnel

Points clés à retenir :

  • Le conditionnel présent = radical du futur + terminaisons de l'imparfait
  • Le conditionnel passé = auxiliaire au conditionnel + participe passé
  • Utilisez-le pour l'hypothèse, la politesse et le futur dans le passé
  • N'oubliez pas l'accord avec le sujet pour les verbes avec être
