Qu'est-ce que le genre et le nombre ?

En français, tous les noms ont un genre (masculin ou féminin) et un nombre (singulier ou pluriel). Ces caractéristiques déterminent la forme des articles et des adjectifs qui les accompagnent.

Exemples :

  • le chat (masculin singulier)
  • la chatte (féminin singulier)
  • les chats (masculin pluriel)
  • les chattes (féminin pluriel)

Les articles définis

Les articles définis (le, la, les) indiquent qu'on parle d'un élément spécifique, connu.

Genre Nombre Article Exemple
Masculin Singulier le le livre
Féminin Singulier la la table
Tous genres Pluriel les les livres, les tables

Cas particuliers

Devant une voyelle :

  • le + voyelle → l' : l'arbre, l'école
  • la + voyelle → l' : l'orange, l'île
  • les reste les : les arbres, les écoles

Le genre des noms

Le genre est souvent arbitraire en français et doit être appris par cœur.

Règles générales

Noms masculins (généralement) :

  • Se terminant par -age : le village, le voyage
  • Se terminant par -ment : le gouvernement, le moment
  • Se terminant par -isme : le capitalisme, le tourisme

Noms féminins (généralement) :

  • Se terminant par -tion : la nation, la situation
  • Se terminant par -sion : la passion, la mission
  • Se terminant par -té : la liberté, la beauté

Exceptions importantes

Attention aux exceptions !

  • Masculins en -tion : le bastion, le cation
  • Féminins en -age : la plage, la cage
  • Féminins en -ment : la jument

Le nombre des noms

Les noms peuvent être au singulier (un seul élément) ou au pluriel (plusieurs éléments).

Formation du pluriel

Règle générale :

On ajoute un -s au singulier.

  • un livre → des livres
  • une maison → des maisons
  • le chat → les chats
  • la table → les tables

Cas particuliers

  • Noms en -s, -x, -z : restent invariables (un bras → des bras)
  • Noms en -al : deviennent -aux (un journal → des journaux)
  • Noms en -eau : deviennent -eaux (un château → des châteaux)

Les accords

Les articles et adjectifs s'accordent avec le nom en genre et en nombre.

Exemples d'accords :

  • le grand livre (masculin singulier)
  • la grande table (féminin singulier)
  • les grands livres (masculin pluriel)
  • les grandes tables (féminin pluriel)

Exercices pratiques

Exercice 1 : Choisissez le bon article

Complétez avec le, la ou les :

  1. ___ livre → le livre
  2. ___ table → la table
  3. ___ enfants → les enfants
  4. ___ arbre → l'arbre
  5. ___ écoles → les écoles

Exercice 2 : Identifiez le genre

Déterminez le genre de ces noms :

  1. la liberté → féminin
  2. le courage → masculin
  3. la beauté → féminin
  4. le gouvernement → masculin
  5. la passion → féminin

Exercice 3 : Formez le pluriel

Mettez au pluriel :

  1. le livre → les livres
  2. la maison → les maisons
  3. le journal → les journaux
  4. le château → les châteaux
  5. le bras → les bras

Exercice 4 : Accordez les adjectifs

Accordez l'adjectif "grand" :

  1. le ___ livre → le grand livre
  2. la ___ table → la grande table
  3. les ___ livres → les grands livres
  4. les ___ tables → les grandes tables

Conseils pour bien utiliser les genres et nombres

Points clés à retenir :

  • Le genre est souvent arbitraire, apprenez-le par cœur
  • Les articles définis s'accordent avec le nom
  • Le pluriel se forme généralement en ajoutant -s
  • Attention aux exceptions (noms en -al, -eau, etc.)
  • Tous les éléments qui accompagnent le nom s'accordent
