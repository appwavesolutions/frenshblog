Qu'est-ce qu'un article ?

L'article est un petit mot qui précède le nom et indique son genre, son nombre et sa détermination. Il est obligatoire devant un nom en français.

Types d'articles :

  • Articles définis : le, la, les (éléments spécifiques)
  • Articles indéfinis : un, une, des (éléments généraux)
  • Articles partitifs : du, de la, des (quantités)

Les articles définis

Les articles définis (le, la, les) indiquent qu'on parle d'un élément spécifique, connu du locuteur et de l'interlocuteur.

Genre Nombre Article Exemple
Masculin Singulier le le livre
Féminin Singulier la la table
Tous genres Pluriel les les livres, les tables

Usage des articles définis

On utilise les articles définis pour :

  • Éléments spécifiques : Le livre que tu m'as prêté
  • Éléments uniques : Le soleil, la lune
  • Généralités : Le chat est un animal
  • Éléments déjà mentionnés : J'ai un chat. Le chat est noir.

Les articles indéfinis

Les articles indéfinis (un, une, des) indiquent qu'on parle d'un élément général, non spécifique.

Genre Nombre Article Exemple
Masculin Singulier un un livre
Féminin Singulier une une table
Tous genres Pluriel des des livres, des tables

Usage des articles indéfinis

On utilise les articles indéfinis pour :

  • Éléments généraux : Un chat dort sur le canapé
  • Première mention : J'ai un chat
  • Quantité indéterminée : Des livres sont sur la table
  • Profession : Il est un médecin

Définis vs Indéfinis

La différence entre articles définis et indéfinis réside dans la spécificité de l'élément dont on parle.

Comparaison :

  • Indéfini : J'ai un chat (n'importe quel chat)
  • Défini : Le chat est noir (le chat dont on parle)
  • Indéfini : Des livres sont sur la table (plusieurs livres)
  • Défini : Les livres sont intéressants (les livres spécifiques)

Cas particuliers

Devant une voyelle

Articles définis :

  • le + voyelle → l' : l'arbre, l'école
  • la + voyelle → l' : l'orange, l'île
  • les reste les : les arbres, les écoles

Articles indéfinis :

  • un reste un : un arbre, un élève
  • une reste une : une orange, une île
  • des reste des : des arbres, des élèves

Absence d'article

Cas où on n'utilise pas d'article :

  • Après "de" : beaucoup de livres
  • Professions : Il est médecin (pas "un médecin")
  • Certaines expressions : avoir faim, avoir soif

Exercices pratiques

Exercice 1 : Choisissez le bon article

Complétez avec le bon article :

  1. ___ livre est intéressant. → Le livre (spécifique)
  2. J'ai ___ chat. → un chat (général)
  3. ___ enfants jouent. → Les enfants (spécifiques)
  4. Il y a ___ table dans la cuisine. → une table (générale)
  5. ___ soleil brille. → Le soleil (unique)

Exercice 2 : Définis ou indéfinis ?

Identifiez le type d'article :

  1. le livre → article défini
  2. une table → article indéfini
  3. les enfants → article défini
  4. un chat → article indéfini
  5. la maison → article défini

Exercice 3 : Devant une voyelle

Complétez devant une voyelle :

  1. ___ arbre → l'arbre
  2. ___ école → l'école
  3. ___ orange → une orange
  4. ___ île → une île
  5. ___ élèves → les élèves

Exercice 4 : Contexte

Choisissez selon le contexte :

  1. J'ai ___ chien. Il s'appelle Rex. → un chien, Le chien
  2. ___ voiture est rouge. → La voiture (spécifique)
  3. Il y a ___ livres sur la table. → des livres (quantité)
  4. ___ lune brille ce soir. → La lune (unique)

Conseils pour bien utiliser les articles

Points clés à retenir :

  • Les articles sont obligatoires devant les noms en français
  • Les articles définis = éléments spécifiques/connus
  • Les articles indéfinis = éléments généraux/nouveaux
  • Attention aux contractions devant les voyelles
  • Le contexte détermine le choix de l'article
