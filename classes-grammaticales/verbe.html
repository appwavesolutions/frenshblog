Qu'est-ce qu'un verbe ?

Le verbe est une classe grammaticale qui exprime une action, un état ou un processus. C'est l'élément central de la phrase qui indique ce qui se passe.

Caractéristiques du verbe :

  • Variable : il se conjugue selon la personne, le nombre, le temps et le mode
  • Noyau de la phrase : il est souvent l'élément principal
  • Régit des compléments : il peut avoir des COD, COI, attributs
  • Indique le temps : présent, passé, futur

Types de verbes

1. Verbes d'action et verbes d'état

Verbes d'action

Ils expriment une action, un mouvement, une transformation.

Exemples :

  • marcher, courir, sauter
  • manger, boire, lire
  • construire, détruire, créer

Verbes d'état

Ils expriment un état, une qualité, une situation.

Exemples :

  • être, paraître, sembler
  • devenir, rester, demeurer
  • avoir l'air, passer pour

2. Verbes transitifs et intransitifs

Verbes transitifs

Ils peuvent avoir un complément d'objet.

  • Transitifs directs : Je mange une pomme. (COD)
  • Transitifs indirects : Je parle à mon ami. (COI)
  • Transitifs doubles : Je donne un livre à Marie. (COD + COI)

Verbes intransitifs

Ils n'ont pas de complément d'objet.

  • Je dors.
  • Il pleut.
  • Nous partons.

Groupes de verbes

Les verbes français sont classés en trois groupes selon leur terminaison à l'infinitif.

1er groupe : Verbes en -er

Le plus grand groupe, environ 90% des verbes français.

Exemples :

  • parler, manger, aimer
  • travailler, étudier, voyager
  • chanter, danser, jouer

2ème groupe : Verbes en -ir

Verbes réguliers en -ir avec participe présent en -issant.

Exemples :

  • finir, choisir, grandir
  • réussir, réfléchir, obéir
  • rougir, jaunir, noircir

3ème groupe : Verbes irréguliers

Verbes avec des conjugaisons particulières.

Exemples :

  • être, avoir, aller
  • faire, venir, prendre
  • vouloir, pouvoir, savoir

Conjugaison

La conjugaison est la variation du verbe selon la personne, le nombre, le temps et le mode.

Les personnes

Personne Singulier Pluriel
1ère personne je nous
2ème personne tu vous
3ème personne il/elle/on ils/elles

Conjugaison du verbe "parler" (1er groupe)

Présent de l'indicatif :

  • Je parle
  • Tu parles
  • Il/Elle/On parle
  • Nous parlons
  • Vous parlez
  • Ils/Elles parlent

Temps et modes

Les modes

Modes personnels :

  • Indicatif : exprime la réalité
  • Subjonctif : exprime le doute, l'émotion
  • Conditionnel : exprime l'hypothèse
  • Impératif : exprime l'ordre, la prière

Modes impersonnels :

  • Infinitif : forme de base du verbe
  • Participe présent : en -ant
  • Participe passé : forme variable
  • Gérondif : en + participe présent

Les temps de l'indicatif

Temps simples :

  • Présent : action actuelle
  • Imparfait : action passée continue
  • Passé simple : action passée ponctuelle
  • Futur simple : action future

Temps composés :

  • Passé composé : action passée révolue
  • Plus-que-parfait : action antérieure
  • Futur antérieur : action future antérieure

Auxiliaires

Les auxiliaires sont des verbes qui aident à former les temps composés.

Être et avoir

Conjugaison de "être" :

  • Je suis
  • Tu es
  • Il/Elle/On est
  • Nous sommes
  • Vous êtes
  • Ils/Elles sont

Conjugaison de "avoir" :

  • J'ai
  • Tu as
  • Il/Elle/On a
  • Nous avons
  • Vous avez
  • Ils/Elles ont

Règle d'accord du participe passé

Avec "être" :

Le participe passé s'accorde avec le sujet.

  • Elle est arrivée. (féminin singulier)
  • Ils sont partis. (masculin pluriel)

Avec "avoir" :

Le participe passé s'accorde avec le COD si celui-ci est placé avant.

  • J'ai mangé une pomme. (COD après)
  • La pomme que j'ai mangée. (COD avant)

Exemples pratiques

Verbes d'action vs verbes d'état

Verbes d'action :

  • Je mange une pomme. (action)
  • Il court dans le parc. (mouvement)
  • Nous construisons une maison. (transformation)

Verbes d'état :

  • Je suis content. (état)
  • Il semble fatigué. (apparence)
  • Nous restons ici. (situation)

Conjugaisons complexes

Verbes irréguliers courants :

  • Aller : je vais, tu vas, il va, nous allons, vous allez, ils vont
  • Faire : je fais, tu fais, il fait, nous faisons, vous faites, ils font
  • Venir : je viens, tu viens, il vient, nous venons, vous venez, ils viennent

Exercices pratiques

Exercice 1 : Identifiez le type de verbe

Classez ces verbes :

  1. manger → verbe d'action, transitif direct
  2. être → verbe d'état, intransitif
  3. parler → verbe d'action, transitif indirect
  4. dormir → verbe d'action, intransitif
  5. donner → verbe d'action, transitif double

Exercice 2 : Conjuguez au présent

Conjuguez "finir" au présent :

  1. Je finis
  2. Tu finis
  3. Il/Elle/On finit
  4. Nous finissons
  5. Vous finissez
  6. Ils/Elles finissent

Exercice 3 : Accord du participe passé

Accordez le participe passé :

  1. Elle est arrivée. (féminin singulier)
  2. Ils sont partis. (masculin pluriel)
  3. La pomme que j'ai mangée. (COD avant, féminin)
  4. J'ai mangé une pomme. (COD après, pas d'accord)

Exercice 4 : Choisissez le bon auxiliaire

Choisissez être ou avoir :

  1. Je suis allé(e) au cinéma. (mouvement)
  2. J'ai mangé une pomme. (action)
  3. Il est né en 1990. (naissance)
  4. Nous avons travaillé dur. (action)

Conseils pour bien utiliser les verbes

Points clés à retenir :

  • Le verbe est le noyau de la phrase
  • Il se conjugue selon la personne, le nombre, le temps et le mode
  • Distinguer verbes d'action et verbes d'état
  • Respecter les règles d'accord du participe passé
  • Choisir le bon auxiliaire (être/avoir)
  • Apprendre les verbes irréguliers par cœur
