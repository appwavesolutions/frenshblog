Qu'est-ce qu'un pronom ?

Le pronom est un mot qui remplace un nom ou un groupe nominal. Il évite les répétitions et permet d'alléger le discours.

Exemple de remplacement :

Sans pronom : "Marie lit un livre. Marie aime ce livre."

Avec pronom : "Marie lit un livre. Elle aime ce livre."

→ Elle remplace "Marie" et ce remplace "un livre"

Types de pronoms

Il existe plusieurs types de pronoms selon leur fonction et leur sens.

Classification des pronoms :

  • Pronoms personnels : je, tu, il, elle, nous, vous, ils, elles
  • Pronoms possessifs : le mien, la tienne, les nôtres...
  • Pronoms démonstratifs : celui, celle, ceux, celles
  • Pronoms relatifs : qui, que, dont, où
  • Pronoms interrogatifs : qui, que, quoi, lequel
  • Pronoms indéfinis : quelqu'un, quelque chose, chacun

Pronoms personnels

Les pronoms personnels remplacent les noms de personnes ou d'objets.

Tableau des pronoms personnels

Personne Sujet COD COI
1ère personne singulier je me me
2ème personne singulier tu te te
3ème personne singulier il/elle/on le/la lui
1ère personne pluriel nous nous nous
2ème personne pluriel vous vous vous
3ème personne pluriel ils/elles les leur

Exemples d'usage

Pronoms sujets :

  • Je mange une pomme.
  • Tu lis un livre.
  • Il travaille.
  • Nous partons demain.

Pronoms COD :

  • Je le vois. (le = le garçon)
  • Tu la connais. (la = la fille)
  • Il nous appelle. (nous = nous)

Pronoms COI :

  • Je lui parle. (lui = à lui/elle)
  • Tu leur écris. (leur = à eux/elles)
  • Il nous téléphone. (nous = à nous)

Pronoms possessifs

Les pronoms possessifs remplacent un nom précédé d'un adjectif possessif.

Tableau des pronoms possessifs

Personne Masculin singulier Féminin singulier Pluriel
1ère personne le mien la mienne les miens/miennes
2ème personne le tien la tienne les tiens/tiennes
3ème personne le sien la sienne les siens/siennes
1ère personne pluriel le nôtre la nôtre les nôtres
2ème personne pluriel le vôtre la vôtre les vôtres
3ème personne pluriel le leur la leur les leurs

Exemples d'usage

Remplacement :

  • Mon livre est rouge. Le tien est bleu. (le tien = ton livre)
  • Sa voiture est neuve. La mienne est ancienne. (la mienne = ma voiture)
  • Nos enfants jouent. Les leurs dorment. (les leurs = leurs enfants)

Pronoms démonstratifs

Les pronoms démonstratifs remplacent un nom précédé d'un adjectif démonstratif.

Tableau des pronoms démonstratifs

Distance Masculin singulier Féminin singulier Pluriel
Proche celui-ci celle-ci ceux-ci
Loin celui-là celle-là ceux-là
Neutre celui celle ceux

Exemples d'usage

Remplacement :

  • Ce livre est intéressant. Celui-ci est ennuyeux. (celui-ci = ce livre-ci)
  • Cette voiture est rapide. Celle-là est lente. (celle-là = cette voiture-là)
  • Ces enfants jouent. Ceux-ci étudient. (ceux-ci = ces enfants-ci)

Pronoms relatifs

Les pronoms relatifs introduisent une proposition subordonnée relative.

Tableau des pronoms relatifs

Pronom Fonction Exemple
qui Sujet L'homme qui parle
que COD Le livre que je lis
dont Complément L'homme dont je parle
où Lieu/Temps La ville où je vis

Exemples d'usage

Propositions relatives :

  • L'homme qui parle est mon père. (qui = sujet de "parle")
  • Le livre que je lis est intéressant. (que = COD de "lis")
  • L'ami dont je te parle est arrivé. (dont = complément de "parle")
  • La ville où je vis est belle. (où = complément de lieu)

Pronoms interrogatifs

Les pronoms interrogatifs servent à poser des questions.

Pronoms interrogatifs simples

  • qui : pour les personnes
  • que/quoi : pour les choses
  • lequel/laquelle/lesquels/lesquelles : pour choisir

Exemples d'usage

Questions :

  • Qui est-ce ? (personne)
  • Que fais-tu ? (chose)
  • Quoi de neuf ? (chose, familier)
  • Lequel préfères-tu ? (choix)

Pronoms indéfinis

Les pronoms indéfinis remplacent un nom de manière imprécise.

Principaux pronoms indéfinis

Personnes :

  • quelqu'un : Quelqu'un frappe à la porte.
  • personne : Personne n'est venu.
  • chacun : Chacun a son opinion.
  • tout le monde : Tout le monde est là.

Choses :

  • quelque chose : J'ai quelque chose à te dire.
  • rien : Il n'y a rien à faire.
  • tout : Tout est prêt.
  • autre chose : Veux-tu autre chose ?

Exercices pratiques

Exercice 1 : Remplacez par un pronom personnel

Remplacez les mots en gras :

  1. Je vois Marie. → Je la vois.
  2. Tu parles à ton ami. → Tu lui parles.
  3. Les enfants jouent. → Ils jouent.
  4. Nous donnons le livre à Paul. → Nous le donnons à lui.

Exercice 2 : Utilisez les pronoms possessifs

Remplacez par un pronom possessif :

  1. Mon livre est rouge. Ton livre est bleu. → Le tien est bleu.
  2. Sa voiture est neuve. Ma voiture est ancienne. → La mienne est ancienne.
  3. Nos enfants jouent. Vos enfants dorment. → Les vôtres dorment.

Exercice 3 : Choisissez le bon pronom relatif

Complétez avec qui, que, dont, où :

  1. L'homme qui parle est mon père.
  2. Le livre que je lis est intéressant.
  3. L'ami dont je te parle est arrivé.
  4. La ville où je vis est belle.

Exercice 4 : Pronoms interrogatifs

Posez des questions avec les pronoms interrogatifs :

  1. Qui est-ce ? (personne)
  2. Que fais-tu ? (chose)
  3. Lequel préfères-tu ? (choix)
  4. Quoi de neuf ? (chose, familier)

Conseils pour bien utiliser les pronoms

Points clés à retenir :

  • Le pronom remplace un nom ou un groupe nominal
  • Il faut respecter l'accord en genre et en nombre
  • Les pronoms personnels ont des formes différentes selon leur fonction
  • Les pronoms relatifs introduisent des propositions subordonnées
  • Attention à la place des pronoms dans la phrase
  • Les pronoms évitent les répétitions et allègent le discours
