Le verbe ÊTRE

Le verbe être est le verbe le plus important du français. Il exprime l'état, l'identité, la profession, la nationalité.

Conjugaison au présent :

  • Je suis
  • Tu es
  • Il/Elle/On est
  • Nous sommes
  • Vous êtes
  • Ils/Elles sont

Usage du verbe être

  • Identité : Je suis Marie
  • Profession : Il est médecin
  • Nationalité : Nous sommes français
  • État : Elle est fatiguée
  • Localisation : Tu es à Paris

Le verbe AVOIR

Le verbe avoir exprime la possession, l'âge, les sensations physiques.

Conjugaison au présent :

  • J'ai
  • Tu as
  • Il/Elle/On a
  • Nous avons
  • Vous avez
  • Ils/Elles ont

Usage du verbe avoir

  • Possession : J'ai une voiture
  • Âge : Il a 25 ans
  • Sensations : J'ai faim, j'ai soif
  • Relations : Elle a deux enfants
  • Caractéristiques : Tu as les yeux bleus

Le verbe ALLER

Le verbe aller exprime le déplacement, le mouvement vers un lieu.

Conjugaison au présent :

  • Je vais
  • Tu vas
  • Il/Elle/On va
  • Nous allons
  • Vous allez
  • Ils/Elles vont

Usage du verbe aller

  • Déplacement : Je vais à Paris
  • Futur proche : Je vais manger
  • Santé : Comment allez-vous ?
  • Direction : Nous allons au cinéma

Le verbe S'APPELER

Le verbe s'appeler exprime le nom, l'identité d'une personne.

Conjugaison au présent :

  • Je m'appelle
  • Tu t'appelles
  • Il/Elle/On s'appelle
  • Nous nous appelons
  • Vous vous appelez
  • Ils/Elles s'appellent

Usage du verbe s'appeler

  • Présentation : Je m'appelle Marie
  • Question : Comment vous appelez-vous ?
  • Identité : Mon fils s'appelle Pierre

Usage pratique

Phrases essentielles

Présentation :

  • Bonjour, je m'appelle Marie.
  • Je suis française.
  • J'ai 25 ans.
  • Je vais à Paris.

Questions courantes

Questions de base :

  • Comment vous appelez-vous ?
  • Quelle est votre nationalité ?
  • Quel âge avez-vous ?
  • Où allez-vous ?

Verbes auxiliaires

Être et avoir comme auxiliaires :

  • Être : Je suis allé(e) au cinéma
  • Avoir : J'ai mangé une pomme
  • Être : Elle est arrivée hier
  • Avoir : Nous avons travaillé

Exercices pratiques

Exercice 1 : Conjuguez ÊTRE

Complétez avec la bonne forme :

  1. Je ___ français. → suis
  2. Tu ___ étudiant. → es
  3. Elle ___ médecin. → est
  4. Nous ___ à Paris. → sommes
  5. Vous ___ fatigués. → êtes

Exercice 2 : Conjuguez AVOIR

Complétez avec la bonne forme :

  1. J'___ une voiture. → ai
  2. Tu ___ 20 ans. → as
  3. Il ___ faim. → a
  4. Nous ___ deux enfants. → avons
  5. Vous ___ soif. → avez

Exercice 3 : Conjuguez ALLER

Complétez avec la bonne forme :

  1. Je ___ au cinéma. → vais
  2. Tu ___ à l'école. → vas
  3. Elle ___ à Paris. → va
  4. Nous ___ au restaurant. → allons
  5. Vous ___ au travail. → allez

Exercice 4 : Conjuguez S'APPELER

Complétez avec la bonne forme :

  1. Je m'___ Marie. → appelle
  2. Tu t'___ Pierre. → appelles
  3. Il s'___ Jean. → appelle
  4. Nous nous ___ Sophie et Paul. → appelons
  5. Vous vous ___ comment ? → appelez

Exercice 5 : Phrases complètes

Complétez les phrases :

  1. Bonjour, je m'___ Marie et je ___ française. → appelle, suis
  2. J'___ 25 ans et je ___ à Paris. → ai, vais
  3. Comment vous ___ ? → appelez-vous
  4. Nous ___ étudiants et nous ___ faim. → sommes, avons

Conseils pour bien utiliser ces verbes

Points clés à retenir :

  • Être et avoir sont les verbes les plus utilisés en français
  • Ils servent aussi d'auxiliaires pour les temps composés
  • Aller exprime le mouvement et le futur proche
  • S'appeler est réfléchi (avec se/s')
  • Ces verbes sont irréguliers, apprenez-les par cœur
