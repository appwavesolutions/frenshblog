Qu'est-ce qu'un pronom sujet ?

Le pronom sujet remplace le nom sujet dans une phrase. Il indique qui fait l'action exprimée par le verbe.

Exemple :

  • Marie mange une pomme. → Elle mange une pomme.
  • Pierre et Paul jouent. → Ils jouent.

Les pronoms sujets

Le français compte 8 pronoms sujets qui correspondent aux différentes personnes grammaticales.

Personne Singulier Pluriel Exemple
1ère personne je nous Je mange / Nous mangeons
2ème personne tu vous Tu manges / Vous mangez
3ème personne il / elle ils / elles Il mange / Ils mangent

Détail des pronoms

1ère personne

  • je : Je mange une pomme.
  • nous : Nous allons au cinéma.

2ème personne

  • tu : Tu es français. (familier)
  • vous : Vous êtes français. (poli/pluriel)

3ème personne

  • il : Il mange. (masculin)
  • elle : Elle mange. (féminin)
  • ils : Ils mangent. (masculin pluriel/mixte)
  • elles : Elles mangent. (féminin pluriel)

Usage des pronoms

Tu vs Vous

Quand utiliser "tu" :

  • Avec la famille, les amis
  • Avec les enfants
  • En situation informelle
  • Entre personnes du même âge

Quand utiliser "vous" :

  • Avec des personnes plus âgées
  • En situation formelle
  • Avec des inconnus
  • Pour plusieurs personnes

Il vs Elle

Accord avec le genre :

  • Il : pour les personnes masculines et les objets masculins
  • Elle : pour les personnes féminines et les objets féminins
  • Exemples : Il est grand. / Elle est grande. / Le livre est sur la table. Il est intéressant.

Ils vs Elles

Règles d'accord :

  • Ils : groupe masculin ou mixte
  • Elles : groupe uniquement féminin
  • Exemples : Pierre et Paul → Ils / Marie et Sophie → Elles / Pierre et Marie → Ils

Accords avec les verbes

Le verbe s'accorde avec le pronom sujet en personne et en nombre.

Conjugaison avec les pronoms

Verbe "manger" au présent :

  • Je mange
  • Tu manges
  • Il/Elle mange
  • Nous mangeons
  • Vous mangez
  • Ils/Elles mangent

Verbes irréguliers

Verbe "être" :

  • Je suis
  • Tu es
  • Il/Elle est
  • Nous sommes
  • Vous êtes
  • Ils/Elles sont

Exercices pratiques

Exercice 1 : Choisissez le bon pronom

Complétez avec le bon pronom :

  1. ___ mange une pomme. (Marie) → Elle
  2. ___ allons au cinéma. (Pierre et moi) → Nous
  3. ___ êtes français ? (à un adulte) → Vous
  4. ___ jouent au football. (les garçons) → Ils
  5. ___ es étudiante. (à une amie) → Tu

Exercice 2 : Accordez le verbe

Conjuguez le verbe "être" :

  1. Je ___ français. → suis
  2. Tu ___ étudiante. → es
  3. Il ___ médecin. → est
  4. Nous ___ à Paris. → sommes
  5. Vous ___ fatigués. → êtes
  6. Ils ___ étudiants. → sont

Exercice 3 : Ils ou Elles ?

Choisissez ils ou elles :

  1. Pierre et Paul ___ jouent. → ils
  2. Marie et Sophie ___ étudient. → elles
  3. Pierre et Marie ___ mangent. → ils
  4. Les filles ___ dansent. → elles
  5. Les garçons ___ courent. → ils

Exercice 4 : Tu ou Vous ?

Choisissez tu ou vous selon le contexte :

  1. ___ es mon ami. → Tu (familier)
  2. ___ êtes mon professeur. → Vous (formel)
  3. ___ as quel âge ? (à un enfant) → Tu
  4. ___ êtes français ? (à un adulte) → Vous
  5. ___ es fatigué ? (à un ami) → Tu

Exercice 5 : Remplacez par un pronom

Remplacez le sujet par un pronom :

  1. Marie mange une pomme. → Elle mange une pomme.
  2. Pierre et Paul jouent. → Ils jouent.
  3. Les filles dansent. → Elles dansent.
  4. Mon frère étudie. → Il étudie.
  5. Mes sœurs chantent. → Elles chantent.

Conseils pour bien utiliser les pronoms sujets

Points clés à retenir :

  • Les pronoms sujets sont obligatoires en français
  • Le verbe s'accorde toujours avec le pronom sujet
  • Choisissez "tu" ou "vous" selon le contexte
  • Utilisez "ils" pour un groupe mixte, "elles" pour un groupe féminin
  • Les pronoms permettent d'éviter les répétitions
