Adjectifs Fréquents pour Décrire Personnes et Objets

Publié le 10 janvier 2024

Apprendre les adjectifs essentiels pour décrire

Niveau : A2 (Élémentaire)

Objectif

Apprendre les adjectifs fréquents pour décrire des personnes et des objets avec précision.

Adjectifs pour les personnes

  • grand / petite - tall / short
  • jeune / vieux - young / old
  • beau / belle - beautiful
  • gentil / gentille - nice / kind
  • intelligent / intelligente - intelligent
  • sympathique - friendly

Adjectifs pour les objets

  • grand / petit - big / small
  • nouveau / nouvelle - new
  • vieux / vieille - old
  • beau / belle - beautiful
  • bon / bonne - good
  • facile / difficile - easy / difficult

Exemples d'utilisation

  • Il est grand et gentil. (He is tall and kind.)
  • C'est un beau livre. (It's a beautiful book.)
  • Elle est intelligente. (She is intelligent.)

Activité pratique

  1. Décrivez 3 personnes avec 3 adjectifs chacune.
  2. Décrivez 3 objets avec des adjectifs.
  3. Créez 5 phrases avec des adjectifs.
