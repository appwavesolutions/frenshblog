Vocabulaire des Sentiments Simples

Publié le 10 janvier 2024

Apprendre : heureux, triste, fatigué

Niveau : A2 (Élémentaire)

Objectif

Apprendre le vocabulaire des sentiments simples pour exprimer vos émotions.

Sentiments positifs

  • heureux / heureuse [uh-RUH / uh-RUHZ] - happy
  • content / contente [kohn-TAHN] - glad / pleased
  • joyeux / joyeuse [zhwa-YUH] - joyful
  • satisfait / satisfaite - satisfied

Sentiments négatifs

  • triste [TREEST] - sad
  • fatigué / fatiguée [fa-tee-GAY] - tired
  • inquiet / inquiète [an-KYAY] - worried
  • fâché / fâchée [fa-SHAY] - angry

Exemples d'utilisation

  • Je suis heureux de vous voir. (I'm happy to see you.)
  • Elle est triste aujourd'hui. (She is sad today.)
  • Je suis fatigué ce soir. (I'm tired this evening.)

Activité pratique

  1. Décrivez comment vous vous sentez aujourd'hui.
  2. Créez 5 phrases sur vos sentiments.
  3. Parlez de ce qui vous rend heureux ou triste.
