Le Conditionnel Présent

Publié le 10 janvier 2024

Premières notions du conditionnel : je voudrais, tu pourrais

Niveau : A2 (Élémentaire)

Plan

  1. Qu'est-ce que le conditionnel ?
  2. Formation du conditionnel présent
  3. Quand utiliser le conditionnel ?
  4. Conclusion

Qu'est-ce que le conditionnel ?

Le conditionnel est un mode qui exprime :

  • Un souhait ou une politesse
  • Une hypothèse
  • Le futur dans le passé

Exemple :

Je voudrais un café, s'il vous plaît.

Le conditionnel exprime ici une politesse (plus poli que "je veux").

Formation du conditionnel présent

Le conditionnel présent se forme avec le radical du futur + les terminaisons de l'imparfait.

Terminaisons du conditionnel :

  • -ais (je)
  • -ais (tu)
  • -ait (il / elle)
  • -ions (nous)
  • -iez (vous)
  • -aient (ils / elles)

Conjugaison du verbe "parler" au conditionnel présent :

  • je parlerais
  • tu parlerais
  • il / elle parlerait
  • nous parlerions
  • vous parleriez
  • ils / elles parleraient

Verbes irréguliers fréquents :

  • être : je serais, tu serais, il serait, nous serions, vous seriez, ils seraient
  • avoir : j'aurais, tu aurais, il aurait, nous aurions, vous auriez, ils auraient
  • aller : j'irais, tu irais, il irait, nous irions, vous iriez, ils iraient
  • vouloir : je voudrais, tu voudrais, il voudrait, nous voudrions, vous voudriez, ils voudraient
  • pouvoir : je pourrais, tu pourrais, il pourrait, nous pourrions, vous pourriez, ils pourraient

Quand utiliser le conditionnel ?

1. Pour exprimer la politesse :

  • Je voudrais un café, s'il vous plaît.
  • Pourriez-vous m'aider ?
  • J'aimerais vous parler.

2. Pour exprimer un souhait :

  • J'aimerais voyager en France.
  • Je voudrais apprendre le français.

3. Pour exprimer une hypothèse :

  • Si j'avais le temps, je viendrais.
  • Si tu étudiais plus, tu réussirais.

Conclusion

Points essentiels à retenir :

  • Le conditionnel = radical du futur + terminaisons de l'imparfait
  • Il exprime la politesse, le souhait ou l'hypothèse
  • "Je voudrais" est plus poli que "je veux"
  • "Pourriez-vous" est plus poli que "pouvez-vous"
  • C'est très utile pour être poli en français
