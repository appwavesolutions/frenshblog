L'Imparfait

Publié le 10 janvier 2024

Apprendre à décrire et parler d'habitudes dans le passé

Niveau : A2 (Élémentaire)

Plan

  1. Qu'est-ce que l'imparfait ?
  2. Formation de l'imparfait
  3. Quand utiliser l'imparfait ?
  4. Conclusion

Qu'est-ce que l'imparfait ?

L'imparfait est un temps du passé qui exprime :

  • Une action en cours dans le passé
  • Une habitude dans le passé
  • Une description dans le passé

Exemple :

Quand j'étais enfant, je jouais au football tous les jours.

L'imparfait exprime ici une habitude dans le passé.

Formation de l'imparfait

L'imparfait se forme à partir de la 1ère personne du pluriel (nous) du présent, en enlevant -ons et en ajoutant les terminaisons de l'imparfait.

Conjugaison du verbe "parler" à l'imparfait :

Pronom Terminaison Conjugaison
je -ais je parlais
tu -ais tu parlais
il / elle -ait il / elle parlait
nous -ions nous parlions
vous -iez vous parliez
ils / elles -aient ils / elles parlaient

Verbe "être" à l'imparfait :

  • j'étais
  • tu étais
  • il / elle était
  • nous étions
  • vous étiez
  • ils / elles étaient

Quand utiliser l'imparfait ?

1. Habitudes dans le passé :

  • Quand j'étais enfant, je jouais au football.
  • Il allait à l'école à pied.

2. Description dans le passé :

  • Il était grand et avait les cheveux bruns.
  • Le ciel était bleu.

3. Action en cours dans le passé :

  • Je marchais dans la rue quand je l'ai vu.
  • Il dormait quand le téléphone a sonné.

Conclusion

Points essentiels à retenir :

  • L'imparfait exprime une habitude, une description ou une action en cours dans le passé
  • Il se forme avec le radical du verbe + terminaisons : -ais, -ais, -ait, -ions, -iez, -aient
  • L'imparfait et le passé composé sont souvent utilisés ensemble
  • L'imparfait = contexte / description, Passé composé = action précise
  • C'est essentiel pour raconter des histoires dans le passé
