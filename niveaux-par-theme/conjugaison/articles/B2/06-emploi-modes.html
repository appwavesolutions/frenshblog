Emploi des Modes selon le Sens

Comprendre quand utiliser l'indicatif, le subjonctif ou le conditionnel

Niveau : B2 (Avancé)

Plan

  1. Les trois modes principaux
  2. L'indicatif
  3. Le subjonctif
  4. Le conditionnel
  5. Conclusion

Les trois modes principaux

En français, il y a trois modes principaux qui expriment différentes intentions :

1. L'indicatif

Exprime la réalité, les faits certains.

  • Je parle français.
  • Il est professeur.

2. Le subjonctif

Exprime la subjectivité, le doute, l'émotion, l'obligation.

  • Il faut que je vienne.
  • Je veux que tu sois là.

3. Le conditionnel

Exprime l'hypothèse, la politesse, le futur dans le passé.

  • Si j'avais le temps, je viendrais.
  • Je voudrais un café.

L'indicatif

L'indicatif est le mode de la réalité et des faits.

Quand utiliser l'indicatif :

  • Pour exprimer des faits réels : Il est français.
  • Pour décrire : Il fait beau.
  • Pour raconter : Il est venu hier.
  • Après "je pense que", "je crois que" (quand c'est certain) : Je pense qu'il viendra.

Le subjonctif

Quand utiliser le subjonctif :

  • Après "il faut que" : Il faut que je vienne.
  • Après "je veux que" : Je veux que tu sois là.
  • Après "bien que" : Bien qu'il soit tard.
  • Après "je doute que" : Je doute qu'il vienne.
  • Pour exprimer l'émotion : Je suis content que tu sois là.

Le conditionnel

Quand utiliser le conditionnel :

  • Pour exprimer l'hypothèse : Si j'avais le temps, je viendrais.
  • Pour la politesse : Je voudrais un café.
  • Pour le futur dans le passé : Il a dit qu'il viendrait.
  • Pour exprimer un regret : J'aurais aimé venir.

Conclusion

Points essentiels à retenir :

  • Indicatif = réalité, faits certains
  • Subjonctif = subjectivité, doute, émotion, obligation
  • Conditionnel = hypothèse, politesse, futur dans le passé
  • Le choix du mode dépend du sens et de l'intention
  • C'est essentiel pour nuancer et argumenter avec précision
