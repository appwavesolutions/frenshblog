Conditionnel Présent et Passé

Publié le 10 janvier 2024

Maîtriser le conditionnel présent et découvrir le conditionnel passé

Niveau : B1 (Intermédiaire)

Plan

  1. Le conditionnel présent
  2. Le conditionnel passé
  3. Quand utiliser le conditionnel ?
  4. Conclusion

Le conditionnel présent

Le conditionnel présent se forme avec le radical du futur + les terminaisons de l'imparfait.

Conjugaison du verbe "parler" :

  • je parlerais
  • tu parlerais
  • il / elle parlerait
  • nous parlerions
  • vous parleriez
  • ils / elles parleraient

Le conditionnel passé

Le conditionnel passé exprime une hypothèse dans le passé ou un regret.

Formation : Conditionnel présent de l'auxiliaire + Participe passé

Avec "avoir" :

  • J'aurais parlé si j'avais eu le temps.
  • Tu aurais mangé si tu avais faim.
  • Il aurait fini son travail.

Avec "être" :

  • Je serais allé(e) si j'avais pu.
  • Tu serais venu(e) plus tôt.
  • Il serait parti avant.

Quand utiliser le conditionnel ?

1. Hypothèses :

  • Si j'avais le temps, je viendrais. (présent)
  • Si j'avais eu le temps, je serais venu. (passé)

2. Regrets :

  • J'aurais aimé venir avec vous.
  • J'aurais voulu te voir.

3. Futur dans le passé :

  • Il a dit qu'il viendrait demain.
  • Elle pensait qu'elle réussirait.

Conclusion

Points essentiels à retenir :

  • Le conditionnel présent = radical du futur + terminaisons de l'imparfait
  • Le conditionnel passé = conditionnel présent de l'auxiliaire + participe passé
  • Il exprime l'hypothèse, le regret ou le futur dans le passé
  • Avec "être", le participe passé s'accorde avec le sujet
  • C'est essentiel pour exprimer des hypothèses complexes
