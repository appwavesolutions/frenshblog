Apprendre à choisir entre indicatif, subjonctif et conditionnel

Niveau : B2 (Avancé)

Plan

  1. L'indicatif
  2. Le subjonctif
  3. Le conditionnel
  4. Comment choisir ?
  5. Conclusion

L'indicatif

L'indicatif exprime la réalité, la certitude, les faits.

Usage de l'indicatif :

  • Pour exprimer des faits réels : Il vient demain.
  • Pour exprimer la certitude : Je sais qu'il vient.
  • Pour décrire la réalité : Il pleut.

Le subjonctif

Le subjonctif exprime le subjonctif, l'émotion, le doute, la volonté.

Usage du subjonctif :

  • Après certaines expressions : Il faut que je parte.
  • Pour exprimer l'émotion : Je suis content qu'il soit là.
  • Pour exprimer le doute : Je doute qu'il vienne.

Le conditionnel

Le conditionnel exprime l'hypothèse, la politesse, le souhait.

Usage du conditionnel :

  • Pour exprimer l'hypothèse : Si j'avais le temps, je voyagerais.
  • Pour la politesse : Je voudrais un café.
  • Pour exprimer le souhait : J'aimerais apprendre.

Comment choisir ?

Critères de choix :

  • Indicatif : certitude, réalité, faits
  • Subjonctif : émotion, doute, volonté, après certaines expressions
  • Conditionnel : hypothèse, politesse, souhait

Conclusion

Points essentiels à retenir :

  • Indicatif = réalité, certitude, faits
  • Subjonctif = émotion, doute, volonté
  • Conditionnel = hypothèse, politesse, souhait
  • Le choix dépend du sens et du contexte
  • Essentiel pour nuancer et exprimer précisément sa pensée
