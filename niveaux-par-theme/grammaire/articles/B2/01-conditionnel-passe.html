Le Conditionnel Passé

Publié le 10 janvier 2024

Apprendre à exprimer un regret ou une hypothèse dans le passé

Niveau : B2 (Avancé)

Plan

  1. Qu'est-ce que le conditionnel passé ?
  2. Comment former le conditionnel passé ?
  3. Quand utiliser le conditionnel passé ?
  4. Exemples de phrases

Qu'est-ce que le conditionnel passé ?

Le conditionnel passé sert à exprimer une hypothèse ou un regret dans le passé.

En bref :

Le conditionnel passé = avoir ou être (au conditionnel présent) + participe passé

Exemple : J'aurais mangé si j'avais eu faim. (I would have eaten if I had been hungry)

Comment former le conditionnel passé ?

Pour former le conditionnel passé, on utilise avoir ou être au conditionnel présent + le participe passé.

Conjugaison avec avoir :

  • j'aurais mangé
  • tu aurais mangé
  • il/elle aurait mangé
  • nous aurions mangé
  • vous auriez mangé
  • ils/elles auraient mangé

Conjugaison avec être :

  • je serais allé(e)
  • tu serais allé(e)
  • il serait allé / elle serait allée
  • nous serions allés(es)
  • vous seriez allé(e)(s)
  • ils seraient allés / elles seraient allées

Quand utiliser le conditionnel passé ?

On utilise le conditionnel passé pour :

  • Exprimer un regret : J'aurais dû étudier.
  • Exprimer une hypothèse dans le passé : Si j'avais eu le temps, j'aurais voyagé.
  • Exprimer une probabilité dans le passé : Il aurait été en retard.
  • Dans le discours rapporté : Il a dit qu'il serait venu.

Exemples de phrases

Regret :

  • J'aurais dû partir plus tôt.
  • Tu aurais pu m'aider.
  • Nous aurions dû étudier davantage.

Hypothèse dans le passé :

  • Si j'avais eu de l'argent, j'aurais voyagé.
  • Si tu avais étudié, tu aurais réussi.
  • Si nous étions partis tôt, nous serions arrivés à l'heure.

Conclusion

Points essentiels à retenir :

  • Le conditionnel passé = avoir/être (conditionnel présent) + participe passé
  • Il exprime regret, hypothèse ou probabilité dans le passé
  • Très utilisé avec "si + plus-que-parfait"
  • Avec être, le participe passé s'accorde avec le sujet
  • Essentiel pour nuancer et exprimer des regrets
