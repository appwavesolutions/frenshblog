Le Plus-que-parfait

Publié le 10 janvier 2024

Apprendre à exprimer une action antérieure à une autre action passée

Niveau : B1 (Intermédiaire)

Plan

  1. Qu'est-ce que le plus-que-parfait ?

  3. Quand utiliser le plus-que-parfait ?

Qu'est-ce que le plus-que-parfait ?

Le plus-que-parfait sert à exprimer une action qui s'est passée avant une autre action passée.

En bref :

Le plus-que-parfait = avoir ou être (à l'imparfait) + participe passé

Exemple : J'avais mangé quand il est arrivé. (I had eaten when he arrived)

Comment former le plus-que-parfait ?

Pour former le plus-que-parfait, on utilise avoir ou être à l'imparfait + le participe passé du verbe.

Conjugaison avec avoir :

  • j'avais mangé
  • tu avais mangé
  • il/elle avait mangé
  • nous avions mangé
  • vous aviez mangé
  • ils/elles avaient mangé

Conjugaison avec être :

  • j'étais allé(e)
  • tu étais allé(e)
  • il était allé / elle était allée
  • nous étions allés(es)
  • vous étiez allé(e)(s)
  • ils étaient allés / elles étaient allées

Quand utiliser le plus-que-parfait ?

On utilise le plus-que-parfait pour :

  • Exprimer une action qui s'est passée avant une autre action passée
  • Raconter des événements dans le passé avec une chronologie
  • Exprimer une action antérieure à une autre

Exemples de phrases

Avec deux actions passées :

  • J'avais mangé (plus-que-parfait) quand il est arrivé (passé composé).
  • Il était parti (plus-que-parfait) avant que je sois arrivé (passé composé).
  • Nous avions fini (plus-que-parfait) nos devoirs quand nos amis sont venus (passé composé).
  • Elle avait étudié (plus-que-parfait) avant de passer (infinitif) l'examen.

Conclusion

Points essentiels à retenir :

  • Le plus-que-parfait = avoir/être (imparfait) + participe passé
  • Il exprime une action antérieure à une autre action passée
  • Essentiel pour raconter des événements avec une chronologie
  • Avec être, le participe passé s'accorde avec le sujet
  • C'est un temps avancé pour raconter des histoires complexes
