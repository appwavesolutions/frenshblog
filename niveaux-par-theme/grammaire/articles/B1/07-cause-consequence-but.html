Cause, Conséquence et But

Publié le 10 janvier 2024

Apprendre parce que, donc, pour exprimer les relations logiques

Niveau : B1 (Intermédiaire)

Plan

  1. La cause (parce que, car, puisque)
  2. La conséquence (donc, alors, ainsi)
  3. Le but (pour, afin de)
  4. Exemples de phrases
  5. Conclusion

La cause (parce que, car, puisque)

Les mots de cause expliquent pourquoi quelque chose se passe.

Les mots de cause :

  • parce que : Je reste à la maison parce que je suis malade.
  • car : Je reste à la maison car je suis malade.
  • puisque : Puisque tu es là, restons ensemble.

La conséquence (donc, alors, ainsi)

Les mots de conséquence expriment le résultat d'une action.

Les mots de conséquence :

  • donc : Il pleut, donc je reste à la maison.
  • alors : Il pleut, alors je reste à la maison.
  • ainsi : Il a étudié, ainsi il a réussi.

Le but (pour, afin de)

Les mots de but expriment l'objectif ou l'intention.

Les mots de but :

  • pour + infinitif : J'étudie pour réussir.
  • pour + nom : J'étudie pour l'examen.
  • afin de + infinitif : J'étudie afin de réussir.

Exemples de phrases

Cause :

  • Je suis fatigué parce que j'ai travaillé beaucoup.
  • Il est absent car il est malade.
  • Puisque tu es là, aidons-nous.

Conséquence :

  • Il a étudié, donc il a réussi.
  • Il pleut, alors nous restons à la maison.
  • Il a travaillé dur, ainsi il a progressé.

But :

  • J'étudie pour réussir l'examen.
  • Il travaille pour gagner de l'argent.
  • Nous partons tôt afin de ne pas être en retard.

Conclusion

Points essentiels à retenir :

  • Cause : parce que, car, puisque (pourquoi)
  • Conséquence : donc, alors, ainsi (résultat)
  • But : pour, afin de (objectif)
  • Ces connecteurs permettent de structurer le discours
  • Essentiels pour argumenter et expliquer
