Apprendre les règles d'accord du participe passé

Qu'est-ce que le participe passé ?

Le participe passé est la forme du verbe utilisée dans les temps composés (passé composé, plus-que-parfait...).

En bref :

Le participe passé s'accorde selon des règles précises avec avoir ou être.

Avec être

Avec être, le participe passé s'accorde toujours avec le sujet.

Exemples avec être :

  • Je suis allé (masculin) / Je suis allée (féminin)
  • Il est arrivé / Elle est arrivée
  • Ils sont partis / Elles sont parties

Avec avoir

Avec avoir, le participe passé s'accorde avec le complément d'objet direct (COD) s'il est placé avant le verbe.

Règle avec avoir :

  • J'ai mangé la pomme. (pas d'accord, COD après)
  • La pomme que j'ai mangée. (accord, COD avant)
  • Les livres que j'ai lus. (accord, COD avant)

Exemples

Avec être :

  • Je suis arrivé.
  • Elle est arrivée.
  • Nous sommes partis.
  • Elles sont parties.

Avec avoir :

  • J'ai mangé la pomme. (pas d'accord)
  • La pomme que j'ai mangée. (accord)
  • J'ai vu les films. (pas d'accord)
  • Les films que j'ai vus. (accord)

Conclusion

Points essentiels à retenir :

  • Avec être : accord toujours avec le sujet
  • Avec avoir : accord avec le COD seulement s'il est avant le verbe
  • C'est une règle complexe mais importante
  • L'accord se fait en genre et en nombre
  • Essentiel pour écrire correctement en français
