Qu'est-ce que le genre ?

En français, tous les noms ont un genre : ils sont soit masculins, soit féminins.

Le genre n'a pas toujours de rapport avec le sexe. Par exemple, "une table" est féminin même si une table n'est ni homme ni femme.

En bref :

Chaque nom en français est soit masculin (le chat), soit féminin (la table).

Il faut apprendre le genre de chaque nom par cœur !

Le masculin

Les noms masculins utilisent l'article "le" ou "un".

Exemples de noms masculins :

  • le chat, un chat
  • le livre, un livre
  • le stylo, un stylo
  • le garçon, un garçon
  • le professeur, un professeur
  • le chien, un chien
  • le soleil, un soleil

Le féminin

Les noms féminins utilisent l'article "la" ou "une".

Exemples de noms féminins :

  • la table, une table
  • la chaise, une chaise
  • la voiture, une voiture
  • la fille, une fille
  • la professeure, une professeure
  • la lune, une lune
  • la maison, une maison

Les articles selon le genre

Tableau récapitulatif :

Genre Article défini Article indéfini Exemple
Masculin le un le chat, un chat
Féminin la une la table, une table

Conclusion

Points essentiels à retenir :

  • Tous les noms en français ont un genre : masculin ou féminin
  • Le masculin utilise "le" ou "un"
  • Le féminin utilise "la" ou "une"
  • Le genre n'a pas toujours de rapport avec le sexe
  • Il faut apprendre le genre de chaque nom par cœur
