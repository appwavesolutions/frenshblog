La cohésion textuelle

La cohésion textuelle assure la liaison logique entre les éléments d'un texte.

Mécanismes de cohésion :

  • Référence : pronoms, déterminants
  • Substitution : remplacer un élément
  • Ellipse : omettre un élément
  • Conjonction : connecteurs logiques
  • Lexique : répétition, synonymes, hyperonymes

Les connecteurs

Les connecteurs organisent le texte et expriment les relations logiques.

Types de connecteurs :

  • Additifs : de plus, en outre, par ailleurs
  • Oppositifs : cependant, néanmoins, toutefois
  • Causaux : en effet, car, puisque
  • Consécutifs : donc, par conséquent, ainsi
  • Temporels : ensuite, puis, enfin
  • Organisateurs : premièrement, d'une part, d'autre part

Structure du discours

La structure du discours organise le texte de manière cohérente.

Éléments de structure :

  • Introduction : présentation du sujet
  • Développement : argumentation structurée
  • Conclusion : synthèse et ouverture
  • Transitions : passages entre les parties
  • Paragraphes : unités thématiques

Exemples

Texte cohérent :

L'apprentissage du français présente de nombreux avantages. En effet, cette langue est parlée dans de nombreux pays. De plus, elle ouvre des perspectives professionnelles intéressantes. Cependant, il faut reconnaître que l'apprentissage demande du temps. Néanmoins, les efforts sont récompensés. Par conséquent, il est recommandé de persévérer.

Conclusion

Points essentiels à retenir :

  • La cohésion textuelle assure la liaison logique
  • Les connecteurs organisent et structurent le texte
  • La structure du discours suit une organisation claire
  • Ces éléments permettent une communication efficace et élégante
  • Essentiels pour maîtriser le français écrit de haut niveau
