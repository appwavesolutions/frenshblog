Types de discours rapporté

Au niveau C2, on maîtrise tous les types de discours rapporté avec leurs nuances.

Les types :

  • Discours direct : Il a dit : "Je viens."
  • Discours indirect : Il a dit qu'il venait.
  • Discours indirect libre : Il marchait. Il viendrait. C'était sûr.
  • Discours narrativisé : Il annonça son arrivée.

Style indirect avancé

Le style indirect avancé maîtrise toutes les transformations et nuances.

Transformations complexes :

  • Concordance des temps dans tous les contextes
  • Transformation des déictiques (ici → là, maintenant → alors)
  • Gestion des modalisations et nuances
  • Style indirect libre avec fluidité

Exemples de phrases

Exemples variés :

  • Direct : Il a dit : "Je viens maintenant."
  • Indirect : Il a dit qu'il venait alors.
  • Indirect libre : Il réfléchissait. Il viendrait. C'était décidé.
  • Narrativisé : Il annonça son arrivée imminente.

Conclusion

Points essentiels à retenir :

  • Maîtrise de tous les types de discours rapporté
  • Précision dans les transformations et concordances
  • Fluidité dans le style indirect libre
  • Essentiel pour une expression littéraire et académique de haut niveau
