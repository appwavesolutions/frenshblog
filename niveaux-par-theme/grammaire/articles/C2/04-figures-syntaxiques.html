Figures Syntaxiques : Inversion, Ellipse, Dislocation

L'inversion

L'inversion change l'ordre habituel des mots pour créer un effet stylistique.

Types d'inversions :

  • Inversion du sujet : Vient le printemps.
  • Inversion après adverbe : Aussi est-il parti.
  • Inversion dans les subordonnées : Quoi qu'il arrive...
  • Inversion stylistique : Rare est le moment où...

L'ellipse

L'ellipse consiste à omettre des mots pour créer un effet stylistique.

Types d'ellipses :

  • Ellipse du verbe : Il est venu, puis [il est] parti.
  • Ellipse du sujet : [Je] Viens ici.
  • Ellipse dans les coordinations : Il a mangé, bu, [il a] dansé.
  • Asyndète : Il est venu, a parlé, parti. (sans conjonction)

La dislocation

La dislocation sépare un élément de la phrase pour le mettre en relief.

Types de dislocations :

  • Dislocation à gauche : Ce livre, je l'ai lu.
  • Dislocation à droite : Je l'ai lu, ce livre.
  • Dislocation double : Ce livre, je l'ai lu, celui-là.

Exemples de phrases

Inversion :

  • Vient le moment de partir.
  • Aussi est-il parti rapidement.
  • Rare est le moment où tout est parfait.

Ellipse :

  • Il est venu, puis [il est] parti.
  • Il a mangé, bu, [il a] dansé.
  • [Je] Viens ici immédiatement.

Dislocation :

  • Ce livre, je l'ai lu plusieurs fois.
  • Je l'ai lu, ce livre.
  • Les étudiants, ils sont arrivés.

Conclusion

Points essentiels à retenir :

  • L'inversion change l'ordre des mots pour un effet stylistique
  • L'ellipse omet des mots pour créer un effet
  • La dislocation met en relief un élément
  • Ces figures syntaxiques permettent une expression sophistiquée
  • Essentielles pour maîtriser le français littéraire et stylistique
