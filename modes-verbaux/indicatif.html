Qu'est-ce que l'indicatif ?

L'indicatif est le mode verbal qui exprime la réalité, les faits, les actions qui se déroulent ou se sont déroulées dans le temps. C'est le mode le plus utilisé en français.

Caractéristiques de l'indicatif :

  • Mode de la réalité : il exprime des faits réels ou considérés comme réels
  • Mode temporel : il situe l'action dans le temps
  • Mode objectif : il présente les faits de manière neutre
  • Mode le plus utilisé : il est présent dans la plupart des phrases

Temps de l'indicatif

L'indicatif comprend huit temps répartis en temps simples et temps composés.

Temps simples

Les 4 temps simples :

  • Présent : action actuelle
  • Imparfait : action passée continue
  • Passé simple : action passée ponctuelle
  • Futur simple : action future

Temps composés

Les 4 temps composés :

  • Passé composé : action passée révolue
  • Plus-que-parfait : action antérieure
  • Passé antérieur : action antérieure au passé simple
  • Futur antérieur : action future antérieure

Présent de l'indicatif

Le présent exprime une action qui se déroule maintenant, une vérité générale, une habitude ou un futur proche.

Conjugaison du verbe "parler"

Personne Conjugaison Exemple
Je parle Je parle français.
Tu parles Tu parles bien.
Il/Elle/On parle Il parle doucement.
Nous parlons Nous parlons ensemble.
Vous parlez Vous parlez fort.
Ils/Elles parlent Ils parlent vite.

Emplois du présent

1. Action actuelle :

  • Je mange une pomme. (maintenant)
  • Il pleut. (en ce moment)

2. Vérité générale :

  • L'eau bout à 100°C.
  • Le soleil se lève à l'est.

3. Habitude :

  • Tous les matins, je prends le bus.
  • Chaque dimanche, nous allons au parc.

Temps du passé

Passé composé

Il exprime une action passée révolue, terminée.

Formation :

Auxiliaire (être/avoir) + participe passé

  • J'ai mangé une pomme.
  • Il est parti hier.
  • Nous avons travaillé dur.

Imparfait

Il exprime une action passée continue, une description, une habitude passée.

Formation :

Radical + terminaisons (-ais, -ais, -ait, -ions, -iez, -aient)

  • Je mangeais une pomme.
  • Il pleuvait hier.
  • Nous travaillions ensemble.

Plus-que-parfait

Il exprime une action antérieure à une autre action passée.

Formation :

Imparfait de l'auxiliaire + participe passé

  • J'avais mangé avant de partir.
  • Il était parti quand j'arrivai.
  • Nous avions travaillé toute la journée.

Temps du futur

Futur simple

Il exprime une action future.

Formation :

Infinitif + terminaisons (-ai, -as, -a, -ons, -ez, -ont)

  • Je mangerai une pomme.
  • Il partira demain.
  • Nous travaillerons ensemble.

Futur antérieur

Il exprime une action future antérieure à une autre action future.

Formation :

Futur simple de l'auxiliaire + participe passé

  • J'aurai mangé avant de partir.
  • Il sera parti quand tu arriveras.
  • Nous aurons travaillé toute la journée.

Emplois de l'indicatif

Dans la narration

Récit au présent :

  • Il entre dans la pièce et voit...
  • Le héros marche vers la forêt.

Dans les descriptions

Description d'un paysage :

  • La montagne s'élève majestueusement.
  • Les oiseaux chantent dans les arbres.

Dans les commentaires

Commentaire sportif :

  • Il marque un but !
  • L'équipe gagne le match.

Exemples pratiques

Dialogue au présent

Conversation :

  • Marie : Que fais-tu ?
  • Pierre : Je lis un livre.
  • Marie : Qu'est-ce que tu penses de ce livre ?
  • Pierre : Il est très intéressant.

Récit au passé

Histoire :

  • Hier, je suis allé au cinéma.
  • Le film était très beau.
  • J'avais déjà vu ce film.
  • Mais je l'ai revu avec plaisir.

Projets futurs

Plans :

  • Demain, je partirai en vacances.
  • Quand j'aurai fini mon travail, je prendrai le train.
  • Je visiterai Paris pendant une semaine.

Exercices pratiques

Exercice 1 : Conjuguez au présent

Conjuguez "manger" au présent :

  1. Je mange
  2. Tu manges
  3. Il/Elle/On mange
  4. Nous mangeons
  5. Vous mangez
  6. Ils/Elles mangent

Exercice 2 : Choisissez le bon temps

Complétez avec le bon temps :

  1. Je mange une pomme. (présent)
  2. Hier, j'ai mangé une pomme. (passé composé)
  3. Demain, je mangerai une pomme. (futur simple)
  4. Quand j'aurai mangé, je partirai. (futur antérieur)

Exercice 3 : Identifiez l'emploi du présent

Identifiez l'emploi :

  1. Je mange une pomme. → action actuelle
  2. L'eau bout à 100°C. → vérité générale
  3. Tous les matins, je prends le bus. → habitude
  4. Demain, je partirai. → futur proche

Exercice 4 : Conjuguez au passé composé

Conjuguez "aller" au passé composé :

  1. Je suis allé(e)
  2. Tu es allé(e)
  3. Il est allé / Elle est allée
  4. Nous sommes allé(e)s
  5. Vous êtes allé(e)s
  6. Ils sont allés / Elles sont allées

Conseils pour bien utiliser l'indicatif

Points clés à retenir :

  • L'indicatif exprime la réalité et les faits
  • Il comprend 8 temps (4 simples + 4 composés)
  • Le présent a plusieurs emplois (actuel, général, habitude)
  • Le passé composé exprime une action terminée
  • L'imparfait exprime une action continue ou une description
  • Le futur simple exprime une action future
  • Respectez les règles d'accord du participe passé
