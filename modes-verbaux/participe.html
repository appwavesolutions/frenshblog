Qu'est-ce que le participe ?

Le participe est une forme verbale qui peut fonctionner comme un adjectif ou comme un verbe. Il y a deux types de participes : le participe présent et le participe passé.

Caractéristiques du participe :

  • Forme verbale : il provient d'un verbe
  • Fonction adjectivale : il peut qualifier un nom
  • Fonction verbale : il peut former des temps composés
  • Variable : il s'accorde selon le contexte

Types de participes

1. Participe présent

Il exprime une action en cours, une qualité ou une cause. Il se termine par -ant.

Exemples :

  • mangeant, buant, dormant
  • parlant, écoutant, regardant
  • allant, venant, partant

2. Participe passé

Il exprime une action accomplie. Il varie selon le verbe et s'accorde en genre et en nombre.

Exemples :

  • mangé, bu, dormi
  • parlé, écouté, regardé
  • allé, venu, parti

Participe présent

Formation du participe présent

Formation : Radical du verbe + -ant

1er groupe : Verbes en -er

Exemples :

  • manger → mangeant
  • parler → parlant
  • aimer → aimant
  • travailler → travaillant

2ème groupe : Verbes en -ir

Exemples :

  • finir → finissant
  • choisir → choisissant
  • grandir → grandissant
  • réussir → réussissant

3ème groupe : Verbes irréguliers

Exemples :

  • être → étant
  • avoir → ayant
  • aller → allant
  • faire → faisant
  • venir → venant

Emplois du participe présent

1. Verbe dans une proposition participiale

Exemples :

  • Mangeant une pomme, je lis un livre.
  • Écoutant la musique, elle danse.
  • Regardant par la fenêtre, il rêve.

2. Adjectif qualificatif

Exemples :

  • une mangeant pomme (qui mange)
  • une dansant fille (qui danse)
  • un courant chien (qui court)

Participe passé

Formation du participe passé

Formation : Radical du verbe + terminaisons spécifiques

1er groupe : Verbes en -er

Exemples :

  • manger → mangé
  • parler → parlé
  • aimer → aimé
  • travailler → travaillé

2ème groupe : Verbes en -ir

Exemples :

  • finir → fini
  • choisir → choisi
  • grandir → grandi
  • réussir → réussi

3ème groupe : Verbes irréguliers

Exemples :

  • être → été
  • avoir → eu
  • aller → allé
  • faire → fait
  • venir → venu

Emplois du participe passé

1. Temps composés

Exemples :

  • J'ai mangé une pomme.
  • Il est parti hier.
  • Nous avons travaillé dur.

2. Adjectif qualificatif

Exemples :

  • une pomme mangée (qui a été mangée)
  • une fille partie (qui est partie)
  • un travail fini (qui est fini)

Accord du participe passé

Avec l'auxiliaire "être"

Le participe passé s'accorde avec le sujet.

Exemples :

  • Elle est arrivée. (féminin singulier)
  • Ils sont partis. (masculin pluriel)
  • Nous sommes allés au cinéma.
  • Vous êtes venus hier.

Avec l'auxiliaire "avoir"

Le participe passé s'accorde avec le COD si celui-ci est placé avant.

Exemples :

  • J'ai mangé une pomme. (COD après, pas d'accord)
  • La pomme que j'ai mangée. (COD avant, accord)
  • J'ai vu mes amis. (COD après, pas d'accord)
  • Mes amis que j'ai vus. (COD avant, accord)

Cas particuliers

Verbes pronominaux :

  • Elle s'est lavée. (accord avec le sujet)
  • Ils se sont téléphoné. (pas d'accord, COD indirect)
  • Nous nous sommes vus. (accord avec le sujet)

Emplois des participes

Participe présent comme adjectif

Exemples :

  • une mangeant pomme (qui mange)
  • une dansant fille (qui danse)
  • un courant chien (qui court)

Participe passé comme adjectif

Exemples :

  • une pomme mangée (qui a été mangée)
  • une fille partie (qui est partie)
  • un travail fini (qui est fini)

Participe dans les temps composés

Exemples :

  • J'ai mangé une pomme.
  • Il est parti hier.
  • Nous avons travaillé dur.
  • Vous êtes venus à temps.

Exemples pratiques

Participe présent

Exemples d'usage :

  • Mangeant une pomme, je lis un livre.
  • Écoutant la musique, elle danse.
  • Regardant par la fenêtre, il rêve.
  • Une mangeant pomme (qui mange)

Participe passé

Exemples d'usage :

  • J'ai mangé une pomme.
  • Il est parti hier.
  • Nous avons travaillé dur.
  • Une pomme mangée (qui a été mangée)

Accord du participe passé

Exemples d'accord :

  • Elle est arrivée. (féminin singulier)
  • Ils sont partis. (masculin pluriel)
  • La pomme que j'ai mangée. (COD avant, accord)
  • J'ai mangé une pomme. (COD après, pas d'accord)

Exercices pratiques

Exercice 1 : Formez le participe présent

Formez le participe présent :

  1. manger → mangeant
  2. parler → parlant
  3. finir → finissant
  4. être → étant

Exercice 2 : Formez le participe passé

Formez le participe passé :

  1. manger → mangé
  2. parler → parlé
  3. finir → fini
  4. être → été

Exercice 3 : Accordez le participe passé

Accordez le participe passé :

  1. Elle est arrivée. (féminin singulier)
  2. Ils sont partis. (masculin pluriel)
  3. La pomme que j'ai mangée. (COD avant, féminin)
  4. J'ai mangé une pomme. (COD après, pas d'accord)

Exercice 4 : Identifiez l'emploi du participe

Identifiez l'emploi :

  1. Mangeant une pomme, je lis. → participe présent, verbe
  2. Une mangeant pomme. → participe présent, adjectif
  3. J'ai mangé une pomme. → participe passé, temps composé
  4. Une pomme mangée. → participe passé, adjectif

Conseils pour bien utiliser les participes

Points clés à retenir :

  • Le participe présent se termine par -ant
  • Le participe passé varie selon le verbe
  • Le participe passé s'accorde avec "être"
  • Le participe passé s'accorde avec "avoir" si le COD est avant
  • Les participes peuvent fonctionner comme des adjectifs
  • Ils forment les temps composés
  • Apprenez les participes irréguliers par cœur
