Qu'est-ce que l'infinitif ?

L'infinitif est la forme de base du verbe, celle qu'on trouve dans le dictionnaire. C'est un mode impersonnel qui ne se conjugue pas et qui peut fonctionner comme un nom.

Caractéristiques de l'infinitif :

  • Mode impersonnel : il ne se conjugue pas
  • Forme de base : c'est la forme du dictionnaire
  • Invariable : il ne change jamais de forme
  • Polyvalent : il peut fonctionner comme un nom

Types d'infinitifs

1. Infinitif présent

Il exprime une action actuelle ou future par rapport au verbe principal.

Exemples :

  • manger, boire, dormir
  • parler, écouter, regarder
  • aller, venir, partir

2. Infinitif passé

Il exprime une action antérieure à celle du verbe principal.

Formation :

  • avoir + participe passé
  • être + participe passé

Exemples :

  • avoir mangé, avoir bu, avoir dormi
  • être parti, être venu, être arrivé

Formation de l'infinitif

Infinitif présent

L'infinitif présent se termine par -er, -ir, -re, -oir selon le groupe du verbe.

1er groupe : Verbes en -er

Exemples :

  • manger, parler, aimer
  • travailler, étudier, voyager
  • chanter, danser, jouer

2ème groupe : Verbes en -ir

Exemples :

  • finir, choisir, grandir
  • réussir, réfléchir, obéir
  • rougir, jaunir, noircir

3ème groupe : Verbes irréguliers

Exemples :

  • être, avoir, aller
  • faire, venir, prendre
  • vouloir, pouvoir, savoir

Emplois de l'infinitif

1. Sujet de la phrase

Exemples :

  • Manger est nécessaire.
  • Étudier demande de la concentration.
  • Voyager enrichit l'esprit.

2. Complément d'objet direct

Exemples :

  • J'aime manger des fruits.
  • Il préfère étudier le soir.
  • Nous aimons voyager en été.

3. Complément d'objet indirect

Exemples :

  • Je pense à partir en vacances.
  • Il rêve de voyager autour du monde.
  • Nous avons envie de manger une glace.

4. Complément circonstanciel

Exemples :

  • Je vais manger au restaurant.
  • Il vient étudier à la bibliothèque.
  • Nous partons voyager demain.

Verbes + infinitif

Verbes de volonté

Exemples :

  • vouloir : Je veux manger.
  • souhaiter : Je souhaite partir.
  • désirer : Je désire voyager.
  • préférer : Je préfère rester.

Verbes de capacité

Exemples :

  • pouvoir : Je peux manger.
  • savoir : Je sais nager.
  • être capable de : Je suis capable de courir.
  • être en mesure de : Je suis en mesure de aider.

Verbes d'émotion

Exemples :

  • aimer : J'aime manger.
  • adorer : J'adore voyager.
  • détester : Je déteste attendre.
  • craindre : Je crains de tomber.

Prépositions + infinitif

Préposition "à"

Exemples :

  • commencer à : Je commence à manger.
  • continuer à : Je continue à étudier.
  • apprendre à : J'apprends à nager.
  • s'habituer à : Je m'habitue à travailler.

Préposition "de"

Exemples :

  • cesser de : Je cesse de fumer.
  • finir de : Je finis de manger.
  • essayer de : J'essaie de comprendre.
  • avoir besoin de : J'ai besoin de dormir.

Préposition "pour"

Exemples :

  • partir pour : Je pars pour voyager.
  • venir pour : Je viens pour t'aider.
  • être là pour : Je suis là pour t'écouter.

Exemples pratiques

Infinitif comme sujet

Exemples :

  • Manger est nécessaire à la vie.
  • Étudier demande de la concentration.
  • Voyager enrichit l'esprit.
  • Lire développe l'imagination.

Infinitif comme complément

Exemples :

  • J'aime manger des fruits.
  • Il préfère étudier le soir.
  • Nous aimons voyager en été.
  • Elle adore lire des romans.

Infinitif avec prépositions

Exemples :

  • Je commence à manger.
  • Je cesse de fumer.
  • Je pars pour voyager.
  • J'ai besoin de dormir.

Exercices pratiques

Exercice 1 : Identifiez l'infinitif

Trouvez l'infinitif dans ces phrases :

  1. Je veux manger une pomme. → manger
  2. Il aime étudier le français. → étudier
  3. Nous préférons voyager en été. → voyager
  4. Elle adore lire des livres. → lire

Exercice 2 : Complétez avec l'infinitif

Complétez avec l'infinitif approprié :

  1. Je veux manger une pomme.
  2. Il aime étudier le français.
  3. Nous préférons voyager en été.
  4. Elle adore lire des livres.

Exercice 3 : Utilisez les prépositions

Complétez avec la bonne préposition :

  1. Je commence à manger.
  2. Je cesse de fumer.
  3. Je pars pour voyager.
  4. J'ai besoin de dormir.

Exercice 4 : Formez l'infinitif passé

Formez l'infinitif passé :

  1. manger → avoir mangé
  2. partir → être parti
  3. venir → être venu
  4. arriver → être arrivé

Conseils pour bien utiliser l'infinitif

Points clés à retenir :

  • L'infinitif est la forme de base du verbe
  • Il ne se conjugue pas et est invariable
  • Il peut fonctionner comme un nom
  • Il s'emploie après de nombreux verbes
  • Il peut être introduit par des prépositions
  • L'infinitif passé exprime l'antériorité
  • Apprenez les verbes qui se construisent avec l'infinitif
